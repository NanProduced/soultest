## 1. 题目

### 1.1 作答形式

- 题型：`双端陈述 + 5 点量表`
- 题量：`32 题`
- 维度：`E/I`、`S/N`、`T/F`、`J/P`
- 交互建议：`一题一卡`、`选后自动进入下一题`
- 单题展示元素：`题号`、`进度`、`题干`、`左右两端陈述`、`5 个选择点`

### 1.2 选项文案与分值映射

| 选项值 | 前台文案 | 计分值 |
| --- | --- | --- |
| 1 | 明显更接近左边 | -2 |

| 3 | 两边差不多 | 0 |
| 4 | 略偏右边 | 1 |
| 5 | 明显更接近右边 | 2 |

## 2. 评分规则

### 2.1 维度归类

- `Q01 ~ Q08` -> `E / I`
- `Q09 ~ Q16` -> `S / N`
- `Q17 ~ Q24` -> `T / F`
- `Q25 ~ Q32` -> `J / P`

### 2.2 计分方式

每题先按选项值映射成 `-2 ~ 2`：

```txt
1 -> -2
2 -> -1
3 ->  0
4 ->  1
5 ->  2
```

四个维度的原始分数分别为：

```txt
EI = Q01 + Q02 + Q03 + Q04 + Q05 + Q06 + Q07 + Q08
SN = Q09 + Q10 + Q11 + Q12 + Q13 + Q14 + Q15 + Q16
TF = Q17 + Q18 + Q19 + Q20 + Q21 + Q22 + Q23 + Q24
JP = Q25 + Q26 + Q27 + Q28 + Q29 + Q30 + Q31 + Q32
```

每个维度的分值范围都是：

```txt
-16 ~ 16
```

### 2.3 四字母判定规则

```txt
EI <= 0 => E
EI >  0 => I

SN <= 0 => S
SN >  0 => N

TF <= 0 => T
TF >  0 => F

JP <= 0 => J
JP >  0 => P
```

### 2.4 结果展示规则

结果页展示：

- `更偏 E / I`
- `更偏 S / N`
- `更偏 T / F`
- `更偏 J / P`
- `最终四字母类型`

结果页不展示：

- `强 INTJ / 中性 INTJ / 临界 INTJ`
- `你有 87% 的 INTJ`
- 认知功能、阴影功能、A/T 子型等超出本测试边界的内容

### 2.5 建议输出结构

## 3. 16 类型长报告版文案（长图增强版）

说明：

- 当前版本在长报告基础上补充了 `关键词`、`第一印象`、`长图标签`，便于后续做长图和结果页模块化排版。
- 文案尽量保持专业测试常见的克制表达，不做绝对化结论。

### 3.1 INTJ｜战略规划者

- 关键词：独立、前瞻、冷静、结构感
- 第一印象：清醒、克制、不容易被外界节奏带跑。
- 结果摘要：你通常先关注事情的整体结构、发展路径和长期影响，再决定是否投入精力。你不太喜欢没有方向感的消耗，更愿意把时间放在真正重要、真正值得的目标上。
- 类型总览：INTJ 往往以独立、克制和有预判力的方式面对世界。你习惯先在心里搭好框架，再逐步验证现实是否符合预期。相比热闹与即时反馈，你更看重逻辑、效率和长期价值，因此容易给人一种冷静、清醒、带有距离感的印象。
- 核心优势：擅长从杂乱信息中抓出主线；能把复杂问题拆解成更可执行的部分；面对长期目标时通常有较强的耐心和持续推进能力。
- 容易卡住：当你过于专注结构和结果时，可能会忽略他人的情绪节奏，也可能低估关系维护本身的重要性。对低效率、反复解释和随意决策的容忍度较低，容易因此显得过于直接或不够柔软。
- 关系中的你：你更看重深度、稳定和价值观上的契合，而不是高频但空转的互动。你不一定会用非常外放的方式表达在意，但一旦确认关系，通常会用持续投入、认真规划和可靠行动来表达重视。
- 工作/学习中的你：你适合需要独立判断、系统思考和长期规划的环境。无论是策略、产品、内容框架、系统搭建还是复杂项目推进，只要问题足够复杂、目标足够清晰，你通常都能发挥优势。
- 高压状态：在高压下，你可能会进一步收紧自己的控制感，对外界的不确定性和低效率变得更敏感。有时你会通过抽离社交、加大自我要求来应对压力，但这也可能让你更难及时释放疲惫。
- 成长提醒：适当让别人看到你的思考过程，而不只是最终结论；在追求正确和有效之外，也给关系中的感受留出位置。
- 长图标签：长期主义、结构脑、清醒克制
- 分享短句：先看清方向，再决定出手。

### 3.2 INTP｜逻辑探索者

- 关键词：理性、好奇、思辨、抽象
- 第一印象：安静但脑内很忙，像是在同时处理好几层问题。
- 结果摘要：你通常会先把问题想通、想透，再决定是否行动。对你来说，真正重要的不只是答案本身，而是这个答案是否经得起推敲。
- 类型总览：INTP 常常表现出独立、理性和高度思辨的一面。你习惯从概念、原理和逻辑关系出发看问题，不轻易接受表面现象，也不喜欢为了迎合环境而快速表态。相比立即达成共识，你更在意内部逻辑是否自洽。
- 核心优势：擅长分析复杂概念；能在别人觉得混乱的地方找到结构和规律；面对新知识时愿意追问“为什么”，而不是只记住表面结论。
- 容易卡住：因为过于在意逻辑质量，你有时会在思考中停留太久，导致行动延后。对流程化、重复性强或需要情绪照顾的事务耐心有限，也可能让别人觉得你难以接近或反应偏冷。
- 关系中的你：你通常更容易通过高质量交流、共同兴趣和思想碰撞建立亲近感。相比频繁互动，你更重视交流本身是否真实、有内容、有意义。
- 工作/学习中的你：你适合研究、分析、策略支持、概念验证、模型搭建这类允许独立思考的工作。只要环境不强迫你过度社交，且能给你一定探索空间，你通常能发挥得更稳定。
- 高压状态：当压力升高时，你可能会进一步退回自己的思考空间，通过反复分析来寻找确定感。短期内这有助于整理思路，但也可能让你更难及时行动或寻求帮助。
- 成长提醒：在追求“想明白”之外，也练习把思考更早地转成行动；不是所有问题都要等到完全确定才开始。
- 长图标签：脑内建模、先想清楚、独立判断
- 分享短句：想透之后，世界才会变清楚。

### 3.3 ENTJ｜目标统筹者

- 关键词：果断、统筹、执行、掌控力
- 第一印象：像一个会自然接过节奏的人，目标感很强。
- 结果摘要：你倾向于快速看清目标、路径和资源分配，并推动事情朝着明确结果前进。对你来说，空转和拖延通常比困难本身更难忍受。
- 类型总览：ENTJ 往往给人干练、果断、有掌控力的印象。你习惯站在更高的位置看局面，迅速判断优先级，并组织行动。你不太满足于停留在讨论层面，而更关心一件事怎样真正被推进、被完成、被做成。
- 核心优势：在判断方向、制定策略和推动执行上通常很有力量；面对复杂局面时能迅速抓住关键变量；容易带动他人一起前进。
- 容易卡住：当结果导向过强时，你可能会低估关系中的缓冲空间，也容易对拖延、犹豫和反复确认感到不耐烦。高标准虽然能提高效率，但也可能让他人感到压力。
- 关系中的你：你欣赏坦诚、独立、有判断力的人。关系里你通常不喜欢过度猜测和含糊表达，更偏好直接沟通、明确需求、一起成长的互动方式。
- 工作/学习中的你：你适合需要统筹、决策、带项目、带节奏的环境。尤其在高复杂度、高目标感、高执行要求的场景中，你往往能比较快进入状态。
- 高压状态：在高压下，你可能更想强化掌控感，对错误、低效和不确定性更敏感。你容易把自己放到持续推进的位置，却忽略身体和情绪也需要恢复。
- 成长提醒：不是所有问题都只能靠更强的推进解决；适度放慢、补足沟通和情绪连接，往往会让结果更稳。
- 长图标签：目标导向、推进力强、先做成再说
- 分享短句：先定方向，再把它做成。

### 3.4 ENTP｜观点开拓者

- 关键词：灵活、发散、机敏、反常规
- 第一印象：脑子转得快，很容易从别人没想到的角度切进来。
- 结果摘要：你擅长从不同角度理解问题，也擅长在固定局面中发现新的解法。很多别人觉得“只能这样”的事，在你看来其实都还有别的可能。
- 类型总览：ENTP 通常思维灵活、反应快、好奇心强，对新点子、新组合和新视角很敏感。你不太喜欢被过早定义，也不满足于按现成规则重复执行。相比守住既有路径，你更习惯打开更多路径。
- 核心优势：有很强的发散能力和重组能力；能迅速发现连接、提出新问法、撬动新机会；面对变化时通常适应得很快。
- 容易卡住：当想法太多、入口太多时，你可能会在探索中分散注意力，导致推进和收束不足。你不喜欢无聊和重复，但现实里很多成果恰恰依赖持续打磨。
- 关系中的你：你通常喜欢有内容、有互动感、能彼此激发的人际关系。你欣赏能接住你想法、愿意一起讨论和探索的人，不太适应长期沉闷、机械的相处状态。
- 工作/学习中的你：你适合创意、策划、早期探索、产品破局、沟通表达等需要快速联想和观点输出的任务。面对从 0 到 1 的问题时，你往往能提供很强的能量。
- 高压状态：在压力下，你可能会通过开启更多可能来回避真正的定夺，表面仍很活跃，内里却逐渐失去聚焦感。
- 成长提醒：保留你的探索天赋，同时练习把最重要的一个想法真正做完；真正的突破，不只来自想法，也来自落地。
- 长图标签：鬼点子多、脑洞在线、不按常规出牌
- 分享短句：看见更多可能，也敢改写旧答案。

### 3.5 INFJ｜洞察协调者

- 关键词：敏锐、深度、共情、意义感
- 第一印象：安静但很有存在感，像是能看见很多没说出口的东西。
- 结果摘要：你往往能在表面信息之外，感知到人、关系和局面中的深层脉络。你不是最喧闹的人，但通常会在安静观察中形成非常完整的理解。
- 类型总览：INFJ 常被认为兼具洞察力与关怀感。你既会关注事情的意义，也会关注人在其中的状态和感受。你不太喜欢浅层、碎片化的互动，更容易被有深度、有方向感、有真实连接的关系与目标吸引。
- 核心优势：擅长理解复杂的人际动态；能把感受、动机和趋势整理成清晰判断；通常具备较强的长期洞察力。
- 容易卡住：因为对人和关系足够敏感，你也更容易被误解、冷淡或长期失衡的关系消耗。很多感受你会先放在心里，表面平静，内里却可能已经很累。
- 关系中的你：你重视稳定、真诚和深度连接，通常不太适应过于表面化或反复试探的互动。你希望关系里不仅有情绪回应，也有价值观和理解层面的契合。
- 工作/学习中的你：你适合需要洞察、理解、表达和长期陪伴的任务，比如内容策划、用户洞察、咨询支持、教育引导、品牌叙事等。

- 成长提醒：行动是你的优势，但适度提前规划会让你的效率更高、代价更低；不是所有问题都需要到了现场才处理。
- 长图标签：行动派、现场感强、反应很快
- 分享短句：先进入场景，再把机会抓住。

### 3.16 ESFP｜活力感染者

- 关键词：热情、亲和、表现力、氛围感
- 第一印象：有温度，也有存在感，很容易把人带进状态。
- 结果摘要：你通常能把现场气氛、人际温度和真实体验带活。你对当下有很强的参与感，也很容易把这种生命力传递出去。
- 类型总览：ESFP 往往自然、亲切、有表现力，重视互动感、真实感和生活本身的鲜活度。你不喜欢冷冰冰地完成任务，更希望人和事情都能有温度、有回应、有参与感。
- 核心优势：通常有较强的亲和力、现场感和表达感染力；无论是在关系里还是在协作中，都容易让氛围变得更轻松、更有人味。
- 容易卡住：因为对外部反馈很敏感，你的状态也更容易被环境影响。面对过度压抑、重复僵化或长期冷淡的场景时，你会比较容易消耗。
- 关系中的你：你重视回应、陪伴、参与感和情绪温度。对你来说，关系不只是“在一起”，更是有没有真正交流、真正感受到彼此。
- 工作/学习中的你：你适合内容表达、活动执行、品牌互动、社群沟通、用户体验等需要人际互动和现场感的工作。
- 高压状态：在高压下，你可能会先用忙碌、热闹或继续投入人群的方式撑住自己，但真正的疲惫往往会在后面慢慢显现。
- 成长提醒：保留你的活力和温度，同时给自己建立一点节奏与边界，让热情不至于总被环境消耗。
- 长图标签：氛围感强、感染力高、现场小太阳
- 分享短句：把温度、活力和参与感带进每个现场。
