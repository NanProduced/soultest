# 九型人格（Enneagram）测试｜题目、评分规则、结果内容文档

> 设计定位：基于 Enneagram Institute、Open Extended Personality Scales（OEPS）开发资料及多个成熟测试平台的公开结构，整理为适合商业产品落地的原创中文 54 题版。
>
> 本文档只保留后续页面设计与测试引擎真正需要的三部分：题目、评分规则、结果内容。研究来源放在末尾附录。
>
> 商业提醒：九型人格并不存在一个可直接照搬的统一免费“官方完整版量表”。当前更稳妥的做法是：参考公开模型与题目结构，采用“概念对齐 + 中文原创改写版”。

## 1. 题目

### 1.1 作答形式

- 适合形态：`5 点量表单选`
- 题干建议：统一写成“以下描述与你的符合程度如何”
- 推荐选项文案：
  - `非常不符合`
  - `比较不符合`
  - `一般`
  - `比较符合`
  - `非常符合`
- 分值映射：`1 / 2 / 3 / 4 / 5`
- 交互建议：
  - 单题大卡片 + 固定底部选项条
  - 不展示实时类型结果
  - 只展示答题进度，不展示类型名称
  - 题目全程混排，避免用户明显感知九型分类套路
  - 54 题版移动端作答时长建议控制在 `7 - 10 分钟`

### 1.2 题量建议

- `27 题`：适合免费引流版，完成率高，但更像初筛，不适合付费长报告
- `36 题`：适合轻量版，能大致分出主型，但翼与近邻类型会偏不稳定
- `54 题`：最适合当前项目首发，每型 `6 题`，信息密度、完成率与结果厚度较平衡
- `90 题+`：更适合后续高客单深度版，不适合作为当前 MVP 首发

### 1.3 九型结构

- `1 型`：改革者 / 完美主义者（Reformer）
  - 关注正确、标准、改进、责任感
- `2 型`：助人者 / 给予者（Helper）
  - 关注关系、被需要、照顾他人、情感连接
- `3 型`：成就者 / 实干者（Achiever）
  - 关注效率、成功、表现、认可与价值感
- `4 型`：个性者 / 浪漫者（Individualist）
  - 关注独特性、真实感、情绪深度与意义感
- `5 型`：观察者 / 探索者（Investigator）
  - 关注知识、边界、理解、独立与掌控消耗
- `6 型`：忠诚者 / 怀疑者（Loyalist）
  - 关注安全、可靠、风险、规则与支持系统
- `7 型`：热情者 / 冒险家（Enthusiast）
  - 关注自由、体验、可能性、快乐与不受限
- `8 型`：挑战者 / 保护者（Challenger）
  - 关注力量、主导权、保护弱者与不被控制
- `9 型`：和平者 / 调停者（Peacemaker）
  - 关注和谐、平稳、少冲突、舒服与内外一致
- 产品表达边界：
  - 九型人格更适合被表述为`核心驱动力模式`
  - 不建议包装成“你天生就是这样、永远不会变”
  - 结果页可强调“当前更像哪一型、哪几型接近”，避免命定论语气

### 1.4 原创中文 54 题题库（混排版）

> 说明：以下题面以“日常行为 + 内在驱动力”组合来写，更适合移动端商业化体验，也更方便后续结果页做细化映射。

1. 我很难对明显的错误、低标准或敷衍了事视而不见。
2. 我习惯先察觉别人的需要，再考虑自己的状态。
3. 我希望自己在别人眼里是高效、优秀、拿得出手的。
4. 我很在意自己有没有活成一个真实而独特的人。
5. 我需要足够多的独处时间和心理空间。
6. 在做决定前，我通常会先想想哪里可能出问题。
7. 一想到新的计划、体验或可能性，我会很快兴奋起来。
8. 当场面需要有人顶上去时，我通常不会躲。
9. 即使我心里有意见，也常会先说“都可以”。
10. 如果一件事还能更好，我会忍不住继续修、继续改。
11. 被人需要会让我觉得自己是有价值的。
12. 比起反复谈感受，我更习惯先把结果做出来。
13. 当生活太普通、太重复时，我会觉得少了点意义。
14. 比起立刻参与，我更习惯先观察、理解、再行动。
15. 面对不确定时，我会本能地寻找可靠的人、规则或依据。
16. 如果一件事让我感到压抑或受限，我会想办法换条路。
17. 我不喜欢被控制、被试探，或被别人拿捏。
18. 我不喜欢高张力的冲突，会本能地想缓和或回避。
19. 我对“应该怎么做”通常有比较清楚的标准。
20. 我不太擅长直接说“我也需要你来照顾我”。
21. 我会留意自己是否在进步、是否被认可。
22. 我常能感受到别人不一定注意到的细腻情绪。
23. 我不喜欢别人突然侵入我的时间、精力或边界。
24. 我对风险和漏洞的敏感度通常比别人高。
25. 我会同时对很多可能性感兴趣，不太想太早收束。
26. 当我在乎的人被冒犯或被欺负，我会很想站出来保护。
27. 我常把别人的事、大家的节奏放在前面，自己的优先级反而靠后。
28. 就算没人要求，我也会想把事情做对、做好、做完整。
29. 当我的付出没有被看见时，我心里会有点委屈。
30. 如果一直没有成绩、没有进展，我会容易怀疑自己的价值。
31. 我会希望自己被真正理解，而不是被简单归类。
32. 当一个问题还没想明白前，我不想轻易表态。
33. 即使表面镇定，我心里也常会先做最坏打算。
34. 我不太喜欢长时间停留在沉重、无聊或痛苦的感受里。
35. 我习惯直接、明确地表达立场，而不是拐弯抹角。
36. 平稳、舒服、没有太多拉扯的状态对我很重要。
37. 我常在心里纠正自己：这里不够好，那样不够对。
38. 我会用照顾、帮忙、体贴来表达感情。
39. 我通常知道怎样让自己显得专业、可靠、有竞争力。
40. 有时候我会反复咀嚼一种情绪或一段关系带来的感受。
41. 我会因为掌握知识、结构或原理而感到安心。
42. 一旦认定一个人或团队值得信任，我会很愿意长期支持。
43. 就算现实有点麻烦，我也会先去想还有什么更有趣的出口。
44. 我宁可自己掌握主动权，也不想把命运交给别人。
45. 我明明心里有想法，却可能拖到很后面才真正表达或行动。
46. 当别人没有原则、没有边界或不讲规则时，我容易不耐烦。
47. 我很在意关系里的温度、回应和彼此之间的靠近感。
48. 为了达到目标，我可以很快调整自己的状态和表现方式。
49. 我既渴望靠近别人，又会担心自己和别人始终隔着一点距离。
50. 情绪太密集、人际太黏，会让我想往后退一点。
51. 我既希望安全稳定，又会反复确认这份安全是不是真的可靠。
52. 我天然想把生活过得有趣、丰富、不被困住。
53. 在冲突里，我往往更容易变得强硬，而不是退让。
54. 我很容易顺着环境走，避免把气氛弄僵。

## 2. 评分规则

### 2.1 九型归类

- `1 型`：1、10、19、28、37、46
- `2 型`：2、11、20、29、38、47
- `3 型`：3、12、21、30、39、48
- `4 型`：4、13、22、31、40、49
- `5 型`：5、14、23、32、41、50
- `6 型`：6、15、24、33、42、51
- `7 型`：7、16、25、34、43、52
- `8 型`：8、17、26、35、44、53
- `9 型`：9、18、27、36、45、54

### 2.2 计分方式

- 每题按 `1 - 5` 计分
- 当前版本建议`不做反向题`
- 每个类型 `6 题`，类型原始分范围为 `6 - 30`
- 建议同时保留：
  - `raw`：原始总分
  - `avg`：平均分
  - `display`：前端展示用百分制映射
  - `band`：低 / 中 / 高分区间

### 2.3 主型判定规则

- 九个类型按 `raw` 从高到低排序
- 最高分类型 = 当前主型候选
- 若第一名与第二名分差 `≤ 2`，结果页建议显示：
  - `主型倾向：1 型 / 9 型接近`
  - 避免强行给出绝对唯一结论
- 若第一名分数明显高于其他类型，可直接展示单一主型
- 对外表达建议使用：
  - `你当前更偏向于 5 型`
  - `你最接近的驱动力模式是 5 型`
  - 尽量少用“你就是……”这种过度绝对的语气

### 2.4 侧翼（Wing）判定规则

> 重要纠正：九型人格里的“侧翼”不是简单的“次高分”，而是主型两侧相邻的类型之一。

- 相邻关系：
  - `1` 的翼只可能是 `9` 或 `2`
  - `2` 的翼只可能是 `1` 或 `3`
  - `3` 的翼只可能是 `2` 或 `4`
  - `4` 的翼只可能是 `3` 或 `5`
  - `5` 的翼只可能是 `4` 或 `6`
  - `6` 的翼只可能是 `5` 或 `7`
  - `7` 的翼只可能是 `6` 或 `8`
  - `8` 的翼只可能是 `7` 或 `9`
  - `9` 的翼只可能是 `8` 或 `1`
- 侧翼建议规则：
  - 先确定主型
  - 再比较主型两侧相邻类型的得分
  - 相邻型中得分更高者 = 翼候选
  - 仅当该翼候选与主型分差 `≤ 5` 时，才展示明确侧翼
  - 若两翼都不高或分差不明显，则显示 `翼倾向不明显`
- 结果页展示示例：
  - `主型：4 型`
  - `翼倾向：4w5`
  - 或 `主型：4 型，翼倾向不明显`

### 2.5 分段建议

- `avg < 2.4`：`low`
- `2.4 ≤ avg < 3.6`：`mid`
- `avg ≥ 3.6`：`high`
- 该分段只用于结果页视觉表达，不建议包装成官方常模或严肃心理测量百分位

### 2.7 结果解释边界

- 九型人格结果更适合被理解为`你习惯依赖的内在驱动力模式`
- 它不是临床诊断，也不是能力高低排名
- 不建议在 MVP 里直接判定：
  - `本能副型（sp / so / sx）`
  - `三型组合（tritype）`
  - `精确成长等级`
- `压力方向 / 成长方向`可以作为结果页补充内容，但不应作为独立判型依据

### 2.8 适合的策略落地方式

- `题目层`：通用答题引擎即可
- `评分层`：`accumulate + ranking + adjacent-wing` 插件最合适
- `结果层`：建议使用`九型人格专用模板`
  - 9 维画像图谱
  - 主型大标题
  - 邻近类型对比
  - 核心动机 / 核心恐惧
  - 关系风格 / 工作风格
  - 压力表现 / 成长建议
- 结论：九型人格适合放进“通用引擎 + 结果模板专用化”的框架内，不需要完全自定义答题流程，但结果页模板应专门设计

## 3. 结果内容

### 3.1 结果页信息架构建议

- `模块 1｜首屏结论`
  - 测试名
  - 用户昵称 / 日期
  - 主型结果
  - 翼倾向（若满足条件）
  - 一句话轮廓总结
- `模块 2｜九型画像图谱`
  - 九型分布图
  - Top3 类型排序
  - 与主型最接近的邻近类型说明
- `模块 3｜你的核心驱动力`
  - 核心渴望
  - 核心恐惧
  - 常见关注点
- `模块 4｜你在人际中的样子`
  - 表达爱与需求的方式
  - 容易产生摩擦的点
  - 更舒服的关系距离
- `模块 5｜你在工作 / 学习中的样子`
  - 优势表现
  - 常见卡点
  - 更适合的环境与协作方式
- `模块 6｜压力与成长`
  - 压力下的防御反应
  - 状态更顺时的成长方向
  - 一条更实际的调节建议
- `模块 7｜长图导出内容`
  - 主型封面标题
  - 结果短句
  - Top3 类型条形图
  - 3 个高共鸣描述点
  - 1 条提醒
  - 1 条长图摘句

### 3.2 9 型长报告文案

#### 1 型｜改革者 / 完美主义者

- `首屏标题`：你当前最接近的驱动力模式是 `1 型改革者`
- `结果摘要`：你通常不是为了“显得厉害”而较真，而是真心希望事情更对、更稳、更像样。你对标准、边界和责任有天然敏感度，看到明显的失误、浪费或不合理，会忍不住想修正。
- `核心驱动力`：想成为一个正确、可靠、有原则的人
- `核心恐惧`：害怕自己犯错、失控、失格，或成为“不够好的人”
- `关系中的你`：你会用认真、负责、替对方考虑周全来表达在意，但也可能因为标准过高而让关系有一点紧。你不是不温柔，只是常把“应该更好”放在前面。
- `工作 / 学习中的你`：你适合规则明确、质量重要、能持续优化的场景。你往往能发现漏洞、压住粗糙、维持秩序，是团队里很可靠的校准者与把关者。
- `压力下的表现`：当事情失控、环境混乱、他人不负责时，你会更容易变得苛刻、紧绷、批判，先盯着问题而不是情绪。
- `成长提醒`：允许自己从“必须完美”退一步，进入“已经足够好，可以先推进”。你真正需要的不是更严厉地逼自己，而是把正确感和松弛感重新放在一起。
- `长图摘句`：你不是太挑剔，你只是很难对“不对劲”装作没看见。

#### 2 型｜助人者 / 给予者

- `首屏标题`：你当前最接近的驱动力模式是 `2 型助人者`
- `结果摘要`：你很容易先看到别人缺什么、难受什么、需要什么。你擅长靠近、照顾、安抚和支持，也常常在“被需要”的感觉里确认自己的价值。
- `核心驱动力`：想通过付出、连接和照顾，成为一个被爱、被看见、对别人重要的人
- `核心恐惧`：害怕自己不被需要、不被爱，或者自己的存在没有被珍惜
- `关系中的你`：你会主动表达关心，记得细节、情绪和需求，关系温度通常来自你的经营。但当自己的需要长期得不到回应时，你也容易委屈，只是未必会直接说。
- `工作 / 学习中的你`：你适合有人际连接、服务、支持、协调、辅导成分的环境。你擅长让别人觉得被理解、被照顾、被接住。
- `压力下的表现`：当付出没有回响时，你可能会更用力地给、更多地介入，甚至在心里累积“为什么没人看见我”的失落。
- `成长提醒`：你不需要靠不断有用，才能换来被爱。比起继续猜别人的需求，更重要的是学会把自己的感受和需要也放上台面。
- `长图摘句`：你表达爱的方式，往往不是说出口，而是提前替别人想到。

#### 3 型｜成就者 / 实干者

- `首屏标题`：你当前最接近的驱动力模式是 `3 型成就者`
- `结果摘要`：你天然会把注意力放在结果、效率、表现和目标感上。你知道怎样把自己调整到“能推进、能完成、能拿成绩”的状态，也希望自己的努力能被看见。
- `核心驱动力`：想成为一个有价值、有成绩、值得被认可的人
- `核心恐惧`：害怕失败、无用、被忽视，或一旦没成果就失去价值感
- `关系中的你`：你往往会先展示自己稳定、能干、靠谱的一面，不太习惯在关系里显露脆弱或混乱。你希望自己是让人放心、值得肯定的存在。
- `工作 / 学习中的你`：你适合节奏明确、目标清晰、反馈直接、结果导向的环境。你能快速进入状态，也擅长包装成果、组织推进和自我驱动。
- `压力下的表现`：当外界反馈下降或目标受阻时，你容易更焦躁地追结果，或者忽视真实疲惫，只靠意志硬顶。
- `成长提醒`：真正稳定的价值感，不应该只靠战绩证明。你可以继续优秀，但也需要允许自己在没有成绩的时候，依然是值得被喜欢的人。
- `长图摘句`：你不是闲不下来，你只是太早学会了用结果证明自己。

#### 4 型｜个性者 / 浪漫者

- `首屏标题`：你当前最接近的驱动力模式是 `4 型个性者`
- `结果摘要`：你会天然关注情绪深度、独特性和真实感。你并不满足于“差不多就行”，你更在意一件事、一段关系、一个表达，是否真正贴近你的内在感受。
- `核心驱动力`：想活得真实、独特、有意义，并被真正理解
- `核心恐惧`：害怕自己平庸、空洞、无足轻重，或永远没有人真正懂你
- `关系中的你`：你渴望深度连接，希望关系里不只是陪伴，还有理解、共鸣和情感质地。你对细腻感受很敏锐，也容易因为落差而受伤。
- `工作 / 学习中的你`：你适合有表达空间、审美空间、个人风格和意义感的任务。机械重复、没有情感投入的工作，容易让你迅速失去状态。
- `压力下的表现`：当你感到被忽视、被误解、被粗暴归类时，你会更容易沉进情绪里，反复感受差异、缺失与不被看见。
- `成长提醒`：深度不是只靠放大情绪获得的，稳定行动也能承载真实。你不需要等到“状态刚刚好”才开始做，你的表达可以在现实中慢慢落地。
- `长图摘句`：你想要的从来不只是热闹，而是真正被看见的那种理解。

#### 5 型｜观察者 / 探索者

- `首屏标题`：你当前最接近的驱动力模式是 `5 型观察者`
- `结果摘要`：你习惯先理解，再靠近；先看清，再投入。你对知识、结构、边界和独立性有天然需求，希望自己的时间、精力和空间能被妥善掌控。
- `核心驱动力`：想通过理解与掌握，维持独立、清醒和不被消耗的状态
- `核心恐惧`：害怕自己无能、空心、被侵入，或被外界过度索取
- `关系中的你`：你不是没有感情，而是更习惯把感情放在有边界、有节奏的位置。你需要信任感，也需要空间感，太快、太密、太黏都会让你后退。
- `工作 / 学习中的你`：你适合分析、研究、系统思考、知识积累、独立解决问题的场景。你能在别人觉得复杂的地方，慢慢搭出理解框架。
- `压力下的表现`：当外界需求过载、人际牵扯过多时，你更容易抽离、关闭、节能，优先保护自己的精力系统。
- `成长提醒`：不是只有完全准备好，才能进入真实世界。知识会给你安全感，但真正的力量也来自适度参与、试错和流动。
- `长图摘句`：你不是冷淡，只是很认真地守着自己的能量边界。

#### 6 型｜忠诚者 / 怀疑者

- `首屏标题`：你当前最接近的驱动力模式是 `6 型忠诚者`
- `结果摘要`：你对风险、不确定性和隐藏问题有天然警觉。你会下意识检查漏洞、评估可靠性、寻找可依赖的人与结构，这种谨慎感既会带来压力，也会带来稳定。
- `核心驱动力`：想确认自己是安全的、被支持的、站在可靠系统里的
- `核心恐惧`：害怕失去依靠、遭遇风险、判断失误，或在关键时刻无人可托底
- `关系中的你`：你很看重忠诚、可靠和一致性。你不会轻易完全放松，但一旦建立信任，就会很投入、很负责，也很愿意长期站在关系里。
- `工作 / 学习中的你`：你适合需要风控意识、预案意识、责任心和团队协同的环境。你常常能比别人更早发现问题、提前规避风险。
- `压力下的表现`：当环境模糊、反馈不稳、规则摇摆时，你会更容易反复确认、想很多、心里拉扯，甚至一边怀疑一边硬撑。
- `成长提醒`：安全感不能只靠外部保证，也来自你对自己判断力的信任。你不需要等到零风险，才允许自己往前走一步。
- `长图摘句`：你不是想太多，你只是太早在脑海里替现实做了风险预演。

#### 7 型｜热情者 / 冒险家

- `首屏标题`：你当前最接近的驱动力模式是 `7 型热情者`
- `结果摘要`：你会天然被新鲜感、自由度、好玩和可能性吸引。你喜欢把生活打开，不想被困在单一、沉重、没有出口的状态里。
- `核心驱动力`：想保持自由、快乐、选择权和面向未来的兴奋感
- `核心恐惧`：害怕受限、停滞、被困住，或长时间停留在痛苦与匮乏里
- `关系中的你`：你常带来轻松、活力、点子和体验感，关系里的快乐值常常被你拉高。但当情绪变重、承诺变硬时，你也可能本能地想找出口。
- `工作 / 学习中的你`：你适合变化快、点子多、空间大、允许探索的场景。你对新机会、新方向、新玩法的感知很快，启动速度也常常很强。
- `压力下的表现`：当现实开始收紧、情绪开始变沉时，你容易靠分散注意、切换目标、寻找新刺激来回避不舒服。
- `成长提醒`：自由不只是不断打开新选项，也包括有能力留在一个选项里，把它走深。你真正想要的，不只是更多可能，而是更完整地拥有其中一种可能。
- `长图摘句`：你看起来总在往前冲，其实是在努力不被困住。

#### 8 型｜挑战者 / 保护者

- `首屏标题`：你当前最接近的驱动力模式是 `8 型挑战者`
- `结果摘要`：你对力量、边界、主动权和真实立场很敏感。你不喜欢被压、被控、被试探，遇到重要的人和事时，也会很自然地进入保护和主导状态。
- `核心驱动力`：想保持力量、掌控感和不被他人左右的主体性
- `核心恐惧`：害怕脆弱、被控制、被伤害，或在关键时刻失去主导权
- `关系中的你`：你往往很直接，很少玩试探和模糊。你在意的人，会明显感受到你的护短、担当与存在感，但也可能感受到你的强势。
- `工作 / 学习中的你`：你适合需要决断、推动、扛事、应对冲突和整合资源的场景。你能在混乱里迅速定调，在关键时刻顶住压力。
- `压力下的表现`：当你感到被限制、被质疑、被拖累时，你会更容易变硬、变急、变得更不愿示弱。
- `成长提醒`：真正稳的力量，不只来自强硬，也来自有能力让别人靠近你的柔软部分。你不需要时时刻刻都站在最前面，才算安全。
- `长图摘句`：你不是脾气大，你只是很难接受自己被动地交出主导权。

#### 9 型｜和平者 / 调停者

- `首屏标题`：你当前最接近的驱动力模式是 `9 型和平者`
- `结果摘要`：你很在意环境里的平稳感、舒服感和少冲突状态。你通常能理解不同立场，也常愿意先让一步，让事情不要变得太僵。
- `核心驱动力`：想维持和谐、内在平稳与关系中的可持续舒适感
- `核心恐惧`：害怕冲突、割裂、失去连接，或因为坚持自己而打破平衡
- `关系中的你`：你容易让人觉得温和、好相处、给人空间。你不喜欢关系里长期拉扯，但也可能因为太常顺着别人，而把自己的真正需要压后。
- `工作 / 学习中的你`：你适合需要协调、整合、维稳、耐心推进的环境。你能包容不同节奏，也能在紧绷的人群里起到缓冲作用。
- `压力下的表现`：当外界要求太多、冲突太近时，你可能会更拖延、更放空、更不想正面碰撞，表面平静，内里却在消耗。
- `成长提醒`：和谐不等于总是退后。真正稳的关系，也容得下你的边界、偏好和不同意。你不需要先把自己变小，才能维持连接。
- `长图摘句`：你不是没有想法，只是太习惯先让气氛保持平静。
