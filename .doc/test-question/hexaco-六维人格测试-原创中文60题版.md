# HEXACO 六维人格测试｜题目、评分规则、结果内容文档

> 设计定位：基于 HEXACO 六维人格模型与官方 60 题 / 100 题结构，整理为适合商业产品落地的原创中文 60 题版。
>
> 本文档只保留后续页面设计与测试引擎真正需要的三部分：题目、评分规则、结果内容。研究来源放在末尾附录。
>
> 商业提醒：若直接使用官方量表原题文本，需先确认授权；当前更建议使用“结构对齐 + 中文原创改写版”。

## 1. 题目

### 1.1 作答形式

- 适合形态：`5 点量表单选`
- 推荐题干语气：第一人称自述句，用户按“像不像我”作答
- 推荐选项文案：
  - `非常不像我`
  - `不太像我`
  - `一般`
  - `比较像我`
  - `非常像我`
- 分值映射：`1 / 2 / 3 / 4 / 5`
- 交互建议：
  - 单题大卡片 + 固定底部选项条
  - 不展示实时结果
  - 只展示进度，不展示维度名称
  - 全程混排六维题目，避免用户感知明显套路
  - 60 题版移动端作答时长建议控制在 `8 - 10 分钟`

### 1.2 题量建议

- `24 题`：适合免费前测版或内容引流版，完成率高，但结果厚度明显不足
- `60 题`：最适合当前项目首发，维度完整、作答负担可控、结果页也足够有内容
- `100 题`：适合后期高客单价 Pro 版，可加更多 facet 解释与更长报告
- `200 题`：更偏研究或重度测评场景，不适合当前商业化 MVP

### 1.3 结构说明

- `H`：诚实—谦逊（Honesty-Humility）
- `E`：情绪性（Emotionality）
- `X`：外向性（Extraversion）
- `A`：宜人性（Agreeableness）
- `C`：尽责性（Conscientiousness）
- `O`：开放性（Openness to Experience）
- HEXACO 的结果应呈现为`六维图谱`，不建议硬收束成单一“人格类型”
- `H` 是 HEXACO 相比大五最有辨识度的维度，结果页应重点丰富这一部分内容

## 2. 评分规则

### 2.1 六维结构

- `H`：诚实—谦逊
  - 侧重：真诚、公平、不过度逐利、不过度炫耀
- `E`：情绪性
  - 侧重：风险敏感、依恋需要、情感触动、脆弱感受阈值
- `X`：外向性
  - 侧重：社交自信、主动表达、活力、互动获得能量
- `A`：宜人性
  - 侧重：宽容、温和、耐心、冲突中的柔软度
- `C`：尽责性
  - 侧重：秩序、勤勉、完成度、谨慎性
- `O`：开放性
  - 侧重：审美、好奇、创意、非传统视角

### 2.2 题目归类与反向计分

#### 诚实—谦逊 H

- 正向题：`1 / 13 / 25 / 37 / 49`
- 反向题：`7 / 19 / 31 / 43 / 55`

#### 情绪性 E

- 正向题：`2 / 14 / 26 / 38 / 50`
- 反向题：`8 / 20 / 32 / 44 / 56`

#### 外向性 X

- 正向题：`3 / 15 / 27 / 39 / 51`
- 反向题：`9 / 21 / 33 / 45 / 57`

#### 宜人性 A

- 正向题：`4 / 16 / 28 / 40 / 52`
- 反向题：`10 / 22 / 34 / 46 / 58`

#### 尽责性 C

- 正向题：`5 / 17 / 29 / 41 / 53`
- 反向题：`11 / 23 / 35 / 47 / 59`

#### 开放性 O

- 正向题：`6 / 18 / 30 / 42 / 54`
- 反向题：`12 / 24 / 36 / 48 / 60`

### 2.3 计分方式

- 每题分值：`1 - 5`
- 反向题换算：`反向得分 = 6 - 原始得分`
- 每个维度总分：该维度所有题目的换算后得分之和
- 每个维度平均分：`维度总分 / 10`
- 推荐结果保留：
  - `raw`：原始总分
  - `avg`：平均分
  - `display`：前端可视化分
  - `band`：高 / 中 / 低档

### 2.4 前端展示分数建议

> 在没有真实常模或授权常模数据时，不建议展示“超过全国 xx% 用户”之类的百分位文案。

- 页面内部可将维度平均分转成视觉指数：
  - `displayScore = ((avg - 1) / 4) * 100`
  - 输出范围：`0 - 100`
- 该指数只用于视觉展示，不等同于常模百分位
- HEXACO 结果页建议采用：
  - `六维雷达图`
  - `六条横向能量条`
  - 或 `六张维度结果卡`

### 2.5 结果分档建议

> 仅用于产品结果文案分层，不宣称为临床 cut-off 或权威常模分界。

- `低区`：`1.0 - 2.4`
- `中区`：`2.5 - 3.5`
- `高区`：`3.6 - 5.0`

### 2.6 结果展示规则

- HEXACO 结果必须展示为`六个独立维度`
- 不建议生成类似 `HAXCO 型`、`六字母人格代号`、`动物人格` 之类二次包装类型
- 首页可重点突出：
  - `最高维度`
  - `第二显著维度`
  - `最低维度`
- `H` 与 `A` 容易被用户混淆，结果页要明确区分：
  - `H` 更偏向公平、真诚、地位与利益取向
  - `A` 更偏向冲突反应、宽容度、脾气与耐心
- `E` 高不等于脆弱，`E` 低也不等于冷漠；结果文案必须避免价值判断
- `60 题版`不建议强做 facet 数值评分；若后期上 `100 题 / 200 题`，再考虑 facet 级结果页

### 2.7 题量建议结论

- 当前项目如果要做首发付费商品，HEXACO 最合适的是`60 题版`
- 原因：
  - 比 24 题更有厚度，结果页不容易显得敷衍
  - 比 100 题更容易完成，移动端流失更低
  - 六维模型本身就比普通五维更有新鲜感，60 题已经足够支撑“可卖、可分享、可复测”的体验
- 如果未来要做高客单价深度版，可以扩成：
  - `HEXACO 100 题 Pro 版`
  - `HEXACO 60 题 + 关系场景加测版`

## 3. 结果内容

### 3.1 结果页信息架构建议

- `模块 1｜首屏结论`
  - 测试名
  - 用户昵称 / 日期
  - 一句话轮廓总结
  - 最高维度 / 第二显著维度 / 最低维度
- `模块 2｜六维图谱`
  - 六维雷达图 / 六条横向条形图
  - 每维一句简述
- `模块 3｜六维长报告`
  - 每个维度单独解释
  - 分高 / 中 / 低三档文案
- `模块 4｜关系与边界`
  - 关系中的信任感
  - 冲突中的处理方式
  - 对亲密与距离的需求
- `模块 5｜合作与决策`
  - 与人协作方式
  - 任务推进节奏
  - 资源、规则、责任上的倾向
- `模块 6｜压力与恢复`
  - 高压下容易出现的状态
  - 对支持、独处、稳定感的需求
- `模块 7｜长图导出内容`
  - 适合分享的封面句
  - 六维图谱
  - 重点维度卡片
  - 一组“别人眼中的你 / 你自己更像怎样”的短句

### 3.2 首屏总结生成规则

- 只描述`维度组合`，不制造新的“人格类型”
- 推荐模板：
  - `你的整体轮廓更偏向「高 {primaryName} + 高 {secondaryName}」，同时在「{lowestName}」维度上相对更克制。`
  - `你在处理关系、规则、情绪和世界的新鲜度时，呈现出比较鲜明的六维偏好。`
- 如果最高维度与第二维度分差小于 `0.2`，建议并列展示
- 如果 `H` 是最高维度，可在首屏单独补一句：
  - `你在公平感、边界感和对“占便宜”的态度上，表现出比较明确的个人原则。`
- 如果 `O` 是最高维度，可在首屏单独补一句：
  - `你对新鲜观点、复杂问题和非标准答案的接受度通常更高。`

### 3.3 六维长报告版文案

#### 3.3.1 诚实—谦逊 H

**高区**

- 你通常比较在意公平、真诚和边界，不太喜欢通过钻空子、过度包装自己或把好处过分往自己身上拉来获得优势。
- 在资源、功劳、利益分配上，你更希望事情“说得过去”，而不是只看自己能拿到多少。
- 你通常不太依赖地位感、炫耀感或优越感来确认自己的价值。
- 别人可能会更容易把“可信”“不爱算计”“相处起来比较放心”这类评价放在你身上。
- 提醒：高 H 不等于什么都让，也不等于不争取；它更像是你在争取时仍然希望方式正当、关系可交代。

**中区**

- 你既会在意公平，也会考虑现实利益。
- 面对规则、资源或功劳分配时，你通常不是绝对理想化，也不是只看自己是否得利。
- 你能理解现实中的博弈，但通常仍会保留基本的分寸感。
- 这种状态会让你在真实社会环境里更灵活，也不至于完全失去原则感。

**低区**

- 你可能更看重实际收益、竞争位置、回报感或优先权，而不是先从“这样做公不公平”出发。
- 在谈判、争取、竞争资源时，你可能更敏锐，也更愿意主动为自己拿结果。
- 在某些高度竞争的环境中，这会成为效率与野心上的优势。
- 但如果短期利益反复压过长期信任，别人可能会逐渐把你理解成“很会算”“不太放心交底”。
- 提醒：H 低不等于人品差，它更多反映的是你对利益、地位和自我优先级的取向；关键在于有没有越过别人对公平的预期。

#### 3.3.2 情绪性 E

**高区**

- 你对风险、失去、关系波动和潜在伤害通常更敏感。
- 你可能更容易被离别、脆弱、危险感或不确定性触发情绪，也更容易希望从重要的人那里得到安慰与回应。
- 你对情感连接的感受阈值通常更低，因此也更容易被真正温柔、可靠的关系打动。
- 这种敏感不是缺点，它也意味着你更容易提前感知问题、更加珍惜人与人之间的依赖与牵挂。
- 提醒：高 E 不等于脆弱，而是说明你的情绪系统对风险与关系信号更敏锐；当缺少恢复机制时，才容易变成反复内耗。

**中区**

- 你会受影响，也能慢慢把自己拉回来。
- 你既不会完全屏蔽情绪，也不至于轻易被每个波动带着走。
- 在关系中，你通常能理解依赖和独立都各有价值。
- 这类状态往往有利于平衡敏感度与稳定度。

**低区**

- 你整体更不容易因为风险、离别、紧张气氛或关系波动而持续陷进去。
- 你通常比较能扛，也不太依赖别人来稳定自己的状态。
- 在高压或不确定环境里，这可能会让你显得更冷静、恢复更快、行动更稳。
- 但在亲密关系里，别人也可能会觉得你太能扛、太不求助、太不容易被看见真实需要。
- 提醒：E 低不等于冷漠，有时只是你习惯把情绪自己消化；但长期不表达，也会让支持资源进不来。

#### 3.3.3 外向性 X

**高区**

- 你通常更容易在社交场合进入状态，也更愿意主动表达、连接和回应外界。
- 与人互动往往会给你带来能量，你也更容易把存在感、活力或带动感传递给周围的人。
- 在陌生环境里，你相对更敢开口、更容易破冰，也更不怕被注意到。
- 别人可能会觉得你有存在感、好接近、反应快、会带气氛。
- 提醒：高 X 不代表你不需要独处；如果长期只维持“外放状态”，也容易忽略自己真正的疲惫。

**中区**

- 你在表达和收回之间通常有比较好的弹性。
- 需要出面时你能站出来，不需要时也能安静退后。
- 你不会为了热闹而热闹，也不至于因为人多就完全退场。
- 这让你在不同类型的关系和场景里都相对好适应。

**低区**

- 你更偏好低刺激、少量但有质量的互动方式。
- 在陌生环境中，你通常需要观察、预热和判断安全感之后，才会逐渐打开。
- 你未必不擅长社交，只是你不习惯把自己持续置于外部中心。
- 在熟悉关系里，你依然可能很真诚、很有趣，只是开启方式比别人更慢。
- 提醒：X 低不等于不合群，但如果你总是不发信号，别人容易误以为你不想靠近。

#### 3.3.4 宜人性 A

**高区**

- 你在冲突里通常更能留余地，不太喜欢把对立推到极致。
- 你更容易原谅、缓和、协商，也更愿意把语气和方式处理得柔和一些。
- 面对冒犯或分歧时，你通常不会第一时间爆回去，而是先压住火气、看有没有更平稳的处理方式。
- 别人可能会觉得你相处起来不刺、比较会给台阶、没那么容易咄咄逼人。
- 提醒：高 A 的代价往往是容易替关系吸收情绪；你可以温和，但不必总是替所有人消化后果。

**中区**

- 你能理解别人，也不会完全失去自己的脾气和立场。
- 面对冲突时，你并非一味退让，但通常也不会轻易把局面推得太僵。
- 这种状态让你在人际里既不显得过硬，也不容易被关系完全裹挟。
- 你可能更倾向于“看人、看事、看边界”地决定要柔到什么程度。

**低区**

- 你在分歧、低效或被冒犯时，可能更容易直接、强硬、急躁，甚至不太愿意把情绪包装得好看。
- 你更像那种会指出问题、推进结论、顶住冲突的人。
- 在需要谈判、维权、止损、强推进的场景里，这会是明显优势。
- 但在人际关系里，如果锋利感总是高于修复意愿，别人可能会逐渐把你体验成“难靠近”“不好惹”“说话太冲”。
- 提醒：A 低不等于刻薄，它更多是在说你对冲突和冒犯的耐受阈值较低；真正决定关系体验的，是你表达锋利时是否仍有分寸。

#### 3.3.5 尽责性 C

**高区**

- 你通常更有秩序感、执行感和完成感。
- 对你来说，把事情安排清楚、推进到位、收尾完整，会比“差不多就行”更重要。
- 你往往更能忍受重复、细节和责任，也更愿意在真正行动前先考虑后果与路径。
- 别人容易把“靠谱”“稳定”“能落地”“不容易掉链子”这些印象放在你身上。
- 提醒：高 C 的一体两面，是容易对自己要求过高，也容易因为计划被打乱而产生额外压力。

**中区**

- 你会在秩序与弹性之间做调节。
- 该认真时能进入状态，遇到变化时也不至于完全僵住。
- 你未必是最严格遵守计划的人，但通常能把关键任务守住。
- 这种状态往往有利于兼顾执行与临场调整。

**低区**

- 你可能更偏向随势而动，而不是事事提前规划。
- 这会让你在变化快、需要临场发挥的场景里更轻盈，也更不容易被流程束住。
- 但在长期任务、细节管理、节奏控制和持续交付上，你可能更容易拖延、分心或失去收尾动力。
- 别人有时并不是觉得你没能力，而是会觉得你不够稳、不够持续、不够让人放心交付。
- 提醒：C 低并不代表懒，而常常意味着你更依赖情境和兴趣驱动；一旦外部结构太弱，结果就会变得不稳定。

#### 3.3.6 开放性 O

**高区**

- 你通常更容易被新观点、新表达、新体验和不止一种答案吸引。
- 你可能对艺术感受、抽象问题、复杂概念、创意连接和非传统想法更有兴趣。
- 面对同一件事，你会天然去想“还有没有别的解释”“有没有另一种可能”。
- 这会让你在理解复杂内容、形成独特视角、建立新连接时更有优势。
- 提醒：高 O 带来的是丰富与延展，但如果缺少收束，也可能表现为分散、跳跃或不够落地。

**中区**

- 你既能接受新东西，也不会过度排斥熟悉路径。
- 面对未知时，你会综合兴趣、现实和成本来判断要不要投入。
- 你不一定追求处处与众不同，但通常保留了理解不同事物的弹性。
- 这让你在创新与执行之间有相对平衡的状态。

**低区**

- 你更偏好清晰、熟悉、可验证的框架，而不是持续探索不确定性。
- 你通常更看重实用性、确定性和现实可操作性，不会为了“新”而“新”。
- 在需要长期执行、流程稳定、标准明确的事情上，这常常是一种优势。
- 但在内容、创意、世界观或新方法上，你也可能更容易对“不合常规”的东西先产生距离感。
- 提醒：O 低不等于没想法，而是在说你更愿意先相信已被证明的路径；只是当环境变化很快时，也要给新可能一点试错空间。

- 如果 `O` 高：`你通常希望关系里能有思想交流、视角碰撞和精神新鲜感。`

#### 模块 4｜成长建议生成逻辑

- 若 `highest = H`：补充 `保持原则感的同时，也允许自己合理争取应得回报。`
- 若 `highest = E`：补充 `敏感不是问题，关键是建立稳定的恢复机制与表达通道。`
- 若 `highest = X`：补充 `把外放能量和真正的休息分开，你会更稳。`
- 若 `highest = A`：补充 `照顾关系很重要，但不必总由你来吸收全部情绪成本。`
- 若 `highest = C`：补充 `结构感是优势，也别让自己被“必须做到最好”绑住。`
- 若 `highest = O`：补充 `保留探索力的同时，把想法拆成可执行步骤会更有成果感。`
- 若 `lowest = H`：补充 `在争取利益时，多问一句“这是否也符合对方的公平预期”。`
- 若 `lowest = E`：补充 `练习适度表达需要，而不是总让别人猜你没事。`
- 若 `lowest = X`：补充 `在重要关系里主动给出一点可见信号，会比沉默更容易被理解。`
- 若 `lowest = A`：补充 `立场可以坚定，但方式不一定只能锋利。`
- 若 `lowest = C`：补充 `把结构外包给工具和流程，通常比单靠自控更有效。`
- 若 `lowest = O`：补充 `在安全范围内给自己一点试错额度，有助于看到新解法。`
