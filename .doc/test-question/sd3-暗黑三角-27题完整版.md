# Short Dark Triad (SD-3) 暗黑三角｜题目、评分规则、结果内容文档

- 更新时间：2026-03-14
- 文档用途：为 `SoulTest` 的测试页、结果页、长图页设计提供可直接落地的内容资产。
- 当前范围：只保留 `题目`、`评分规则`、`结果内容` 三部分；研究来源放在文末附录。
- 使用原则：SD-3 是 `三维人格特质量表`，不是类型测试，也不是医学或临床诊断工具。结果文案应围绕“倾向”来写，避免道德审判化、病理化、命定化表达。

## 1. 题目

### 1.1 作答形式

- 题型：`单句陈述 + 5 点同意度量表`
- 题量：`27 题`
- 维度：`马基雅维利主义`、`自恋`、`精神病态`
- 交互建议：`一题一卡`、`选后自动进入下一题`、`顶部固定进度条`
- 单题展示元素：`题号`、`进度`、`陈述句题干`、`5 个同意度选项`
- 题面风格建议：保持克制、直给、少解释，避免在题面加入引导性说明
- 运营提示：`Q26` 含性行为表述，若前台分发场景对审核较敏感，建议保留研究版原意，同时准备一套审核兼容的替换题面用于渠道投放

### 1.2 选项文案与分值映射

| 1 | 完全不同意 | 1 |
| 2 | 比较不同意 | 2 |

## 2. 评分规则

### 2.1 维度归类

- `Q01-Q09` => `马基雅维利主义`
- `Q10-Q18` => `自恋`
- `Q19-Q27` => `精神病态`

### 2.2 反向计分题

- 反向题：`Q11`、`Q15`、`Q17`、`Q20`、`Q25`
- 反向换算：`reverseScore = 6 - rawScore`

示例：

- 用户选 `1` => 反向后记为 `5`
- 用户选 `2` => 反向后记为 `4`
- 用户选 `3` => 反向后仍为 `3`
- 用户选 `4` => 反向后记为 `2`
- 用户选 `5` => 反向后记为 `1`

### 2.3 计分方式

每题得分为 `1~5`，先处理反向题，再计算三条维度均分：

```text
马基雅维利主义 = avg(Q01, Q02, Q03, Q04, Q05, Q06, Q07, Q08, Q09)
自恋 = avg(Q10, R11, Q12, Q13, Q14, R15, Q16, R17, Q18)
精神病态 = avg(Q19, R20, Q21, Q22, Q23, Q24, R25, Q26, Q27)
```

其中：

- `avg` = 平均分
- `Rxx` = 该题先做反向计分后再参与平均
- 三个维度最终都落在 `1.00 ~ 5.00`

### 2.4 主导维度判定

- `primaryTrait` = 三个维度中分数最高的一项
- 若第一名与第二名分差 `<= 0.25`，建议前台展示为 `双高混合倾向`，不要强行只给一个主标签
- 若三维都接近中段，建议展示为 `整体分布较均衡`，避免硬贴极端人设

### 2.5 分档建议（产品展示用，不宣称为官方 cut-off）

说明：

- 原始 SD-3 是 `连续维度量表`，没有统一的临床式“正常 / 异常”官方阈值。
- 对消费级产品，MVP 阶段可先用三档文案切换；上线后更建议结合站内样本做 `百分位` 展示。

临时三档建议：

- `1.00 ~ 2.19` => `较低`
- `2.20 ~ 3.39` => `中段`
- `3.40 ~ 5.00` => `较高`

### 2.6 结果展示规则

建议结果页固定输出以下信息：

- `三维分数`：马基雅维利主义 / 自恋 / 精神病态
- `主导维度`：最高分维度或双高混合维度
- `雷达图 / 三柱图`：用于直观看三维结构
- `维度拆解文案`：每一维按低 / 中 / 高切换对应说明
- `主结果摘要`：围绕主导维度输出一句到一段的总评
- `边界提醒`：明确“这是人格倾向，不是诊断，不等于道德评价”

## 3. 结果内容

### 3.1 结果页信息架构建议

#### 3.1.1 页面目标

- 第一眼就让用户知道：`哪一维最突出`、`整体结构长什么样`
- 第二屏开始让用户感觉：`这份结果不是一句标签，而是对自己处理关系与竞争方式的拆解`
- 页面末尾要自然承接：`保存长图`、`分享结果`、`去看别的测试`
- 内容密度要做到：`首屏能秒懂`，`下滑后觉得内容足够值回票价`

#### 3.1.2 推荐页面结构（移动端优先）

1. `Hero 结论首屏`
2. `三维图谱总览`
3. `主导维度深读`
4. `另外两维补充解释`
5. `你在人际关系中的处理方式`
6. `你在竞争 / 利益场景中的风格`
7. `这份倾向最容易带来的代价`
8. `边界提醒 + 分享 / 保存 CTA`

#### 3.1.3 首屏 Hero 必须包含的信息

- `主标题`：直接给主导倾向判断
- `副标题`：一句更像人话的解释，不写学术名词堆砌
- `主标签`：如 `策略操盘倾向偏高` / `聚光主场倾向偏高`
- `次标签`：展示第二高维度，避免结果过于扁平
- `三维分数概览`：建议用 `雷达图 + 数值标签` 或 `横向三柱`
- `一句短评`：让用户快速形成“这说的就是我”的第一印象
- `分享动作`：`保存结果`、`生成长图`、`再测一套`

#### 3.1.4 三维总览区建议

这一屏的任务不是解释全部结论，而是让用户先看懂结构。

建议包含：

- `三维排序`：从高到低排列 `马基雅维利主义 / 自恋 / 精神病态`
- `维度差值`：补充第一名与第二名是否接近
- `可视化组件`：优先 `雷达图`，备选 `横向条形图`
- `数据标签`：每维显示 `分数 + 分档`
- `一句总述`：例如 `你的结构不是单点突出，而是“策略感 + 尊重需求”同时在线。`

图表建议：

- 主图：`Radar / Spider Chart`
- 备选图：`Horizontal Bar`
- 无障碍补充：图下方必须同步给出 `文字版分数表`

#### 3.1.5 主导维度深读区建议

这是结果页最值钱的一屏，建议单独做成视觉最强的一张卡。

必须包含：

- `这一维为什么高`：把抽象特质翻译成日常行为
- `你最像哪类人`：不是给人格类型，而是给处理问题的方式
- `这一维在什么场景最明显`：关系、竞争、冲突、被忽视、被压制等
- `优势面`：为什么这种倾向在某些环境里有力量
- `代价面`：为什么它也会推高关系和后果成本

文案要求：

- 先给 `熟悉感`
- 再给 `解释感`
- 最后给 `边界感`

#### 3.1.6 另外两维补充区建议

不要只写主导维度，否则用户会觉得信息单薄。

建议每一维都用一张小卡展示：

- `维度名称`
- `当前分档`
- `一句概述`
- `这个分档下最常见的表现`
- `这个分档下最容易忽视的问题`

这样结果页会从“单标签”升级为“完整结构报告”。

#### 3.1.7 场景化模块建议

结果页中后段建议至少补两类场景：

- `在人际关系中的你`
- `在竞争 / 利益场景中的你`

如果要继续拉高内容厚度，可再补：

- `当你感到被冒犯时`
- `当你想赢下局面时`
- `当你遇到更强势的人时`

这些模块最适合做成 `2~3 行短段落 + 关键词标签`，不宜写得像学术解释。

#### 3.1.8 页面尾段建议

尾段不只是免责声明，还要承担转化任务。

建议包含：

- `风险提醒`：高分不等于“坏”，低分也不等于“好”
- `边界说明`：这是人格倾向测量，不是诊断结果
- `分享短句`
- `保存长图按钮`
- `返回首页继续逛题集`

#### 3.1.9 结果页文案密度建议

- 首屏短评：`35~55 字`
- 单卡正文：`60~110 字`
- 单模块总字数：尽量控制在 `120~180 字`
- 长内容尽量拆为：`一句总述 + 2~3 个要点`
- 避免整屏都是大段文字，影响用户继续滑动


### 3.2 维度长报告文案

#### 3.2.1 马基雅维利主义（前台副标题：策略操盘倾向）

- 关键词：策略、算计、利益优先、控制信息
- 核心定义：更关注局势、筹码和获利路径，倾向于用更现实、更工具化的方式处理关系与资源。

**较低**

- 结果摘要：你通常不太习惯把人际关系当成博弈，也不太愿意为了结果刻意布局或隐藏信息。
- 人际中的你：你更容易用直接、真诚的方式和人来往，不太擅长也不太喜欢绕弯子。
- 利益/竞争中的你：你会更在意过程是否坦荡，而不是“有没有更聪明的手段”。
- 可能代价：在复杂人情和资源博弈里，你有时会显得过于直给，容易低估别人对信息、关系和时机的经营。
- 长图标签：不爱算计、偏真诚、关系直给

**中段**

- 结果摘要：你并不是时时都在“算”，但在关键场景里，你知道什么时候该观察局势、保留信息、优先保护自己。
- 人际中的你：日常相处并不一定强势，但你会在需要时拉开边界，不轻易把底牌全部摊开。
- 利益/竞争中的你：你懂得看人下菜碟，也会权衡“现在说什么、做什么，对自己最有利”。
- 可能代价：如果长期处于高竞争环境，容易逐渐把关系也视为资源交换，亲密感会被策略感稀释。
- 长图标签：会看局势、懂得保留、先想后动

**较高**

- 结果摘要：你很容易从“局势、筹码、时机、收益”这些角度理解人与事，直觉上会先判断怎么做更有利。
- 人际中的你：你未必外显强势，但通常不会轻易暴露真实想法；你更擅长控节奏，而不是交底牌。
- 利益/竞争中的你：你对权力关系、信息差和关键人物很敏感，知道什么时候该退、该忍、该出手。
- 可能代价：当策略感过强时，别人可能会感到被试探、被防备或被利用；长期如此，关系的信任成本会变高。
- 风险提醒：高马基雅维利倾向不等于“成熟城府”，也可能意味着过度防御、过度工具化和对关系安全感不足。
- 长图标签：先看筹码、擅长控局、很会留后手
- 分享短句：你更像一个先看局势，再决定怎么走的人。

#### 3.2.2 自恋（前台副标题：聚光主场倾向）

- 关键词：自我放大、尊重需求、舞台感、优越感
- 核心定义：更在意自我形象、被看见、被认可和被尊重，希望自己的存在感与价值感被外界确认。

**较低**

- 结果摘要：你通常不太主动把自己放到舞台中央，也不会强烈追求成为被注目的那个人。
- 人际中的你：你更重视自在、自然、不端着的相处，不太喜欢刻意经营个人光环。
- 利益/竞争中的你：你可能不是最爱“出头”的那类人，更习惯先把事情做好，而不是先把自己放大。
- 可能代价：当机会需要主动争取、主动曝光时，你可能显得过于收敛，甚至低估自己的价值。
- 长图标签：不抢镜、较低表现欲、存在感自然

**中段**

- 结果摘要：你需要被尊重，也愿意在合适的时候展现自己，但不一定时时都要站在聚光灯下。
- 人际中的你：你希望自己的价值被看到，也会在被忽视或被轻慢时明显感到不舒服。
- 利益/竞争中的你：你有一定表现欲和向上意识，知道适度展示自己能换来更好的位置和资源。
- 可能代价：如果外界反馈不稳定，你可能会在自信和自我怀疑之间来回摆动。
- 长图标签：需要被看见、重视认可、懂得展示自己

**较高**

- 结果摘要：你对“我值不值得被看见、被认可、被尊重”这件事格外敏感，也更容易把自己放在舞台中心理解世界。
- 人际中的你：你通常有较强的存在感，希望别人注意到你的能力、气场或特别之处，不太接受被随意忽略。
- 利益/竞争中的你：你更愿意争取高位、头衔、资源和关注，也更善于塑造自己的可见度与重要性。
- 可能代价：如果长期需要靠外界反馈维持自我评价，容易在赞赏不足时感到失衡、受挫或被冒犯。
- 风险提醒：高自恋倾向不等于真正稳定的自信；有时恰恰说明对外部认可的依赖更强。
- 长图标签：需要主场、讨厌被忽视、很在意尊重
- 分享短句：你不是想随便被看见，而是想被认真看见。

#### 3.2.3 精神病态（前台副标题：冷感冒险倾向）

- 关键词：冷感、冲动、报复、低共情、风险偏好
- 核心定义：更低的恐惧感和更高的刺激寻求，伴随更低的内疚、更多的冲动性，以及对后果的钝感。

**较低**

- 结果摘要：你通常会更自然地避开高风险、高冲突和高破坏性的行为，对后果也更有顾忌。
- 人际中的你：你不太容易把关系推到失控边缘，也不太会把“伤人”当成无所谓的事情。
- 利益/竞争中的你：你通常愿意踩刹车，先看代价，再决定要不要做得太绝。
- 可能代价：在极端竞争或高压局面里，你可能因为顾虑过多而显得不够狠、不够快。
- 长图标签：顾及后果、风险偏好低、不轻易失控

**中段**

- 结果摘要：你并不是一个完全冒进的人，但在被激怒、被逼迫或高刺激情境里，可能会表现出更硬、更冲、更不想退让的一面。
- 人际中的你：平时未必明显，但一旦踩到底线，你会更倾向于直接反击，而不是慢慢消化。
- 利益/竞争中的你：你有一定狠劲，必要时能把姿态放硬，但并不总是以破坏性方式解决问题。
- 可能代价：如果情绪调节和后果评估不足，容易在短时冲动里做出事后要收拾残局的决定。
- 长图标签：有点狠劲、踩线会反击、不太怕硬碰硬

**较高**

- 结果摘要：你对危险、冲突和刺激的敏感度可能更低，也更容易在高压或对抗中采取强硬、冒险、甚至带攻击性的处理方式。
- 人际中的你：你不太喜欢示弱，也不太愿意被拿捏；一旦被挑衅，往往更倾向立刻反制。
- 利益/竞争中的你：你行动果断，敢压、敢顶、敢赌，很多人不敢做的决定你反而更敢拍板。
- 可能代价：当冲动、报复心和低后果敏感叠在一起时，很容易在短期“赢一口气”的同时，拉高长期代价。
- 风险提醒：这里的“精神病态”是量表维度名，不等于临床诊断；对前台更建议配合副标题 `冷感冒险倾向` 一起展示。
- 长图标签：不怕硬碰、反击很快、后果感偏低
- 分享短句：你更像一个不爱退、也不太怕翻桌的人。

### 3.3 主导维度头图文案

#### 马基雅维利主义主导

- 主标题：你的人际雷达，更偏向“先看局势再行动”
- 一句话摘要：你会先判断谁重要、信息值不值钱、时机对不对，再决定自己怎么出手。
- 首屏短评：比起情绪化反应，你更在意结构、筹码和结果。你不一定最外放，但通常很清楚什么该说、什么不该说。

#### 自恋主导

- 主标题：你的自我雷达，更偏向“需要被看见与被尊重”
- 一句话摘要：你会更自然地关注自己是否被认可、是否有分量、是否站在该属于自己的位置上。
- 首屏短评：你不是单纯爱表现，而是更难接受自己的价值被忽略。对你来说，被看见和被尊重，本身就是重要的心理需求。

#### 精神病态主导

- 主标题：你的应对方式，更偏向“硬、快、敢冒险”
- 一句话摘要：面对冲突、挑衅或强压时，你更可能选择直接出手，而不是慢慢退让。
- 首屏短评：你对风险和压迫的耐受度可能更高，也更容易在高刺激场景里做出强硬决策；这会带来气场，也会抬高代价。

#### 双高混合倾向

- 主标题：你的暗黑侧面，不只落在一个点上
- 一句话摘要：你的前三维里，有两项同时更突出，所以你在关系与竞争里不会只呈现一种单线风格。
- 首屏短评：你既可能会算，也可能会要面子；既可能会强硬，也可能很会控局。比起单一标签，你更适合看完整结构。

#### 双高组合补充导语

- `马基雅维利主义 + 自恋`：你既在意位置，也懂得运作局面。很多时候你不是简单想赢，而是想在赢的同时保持自己在场上的分量。
- `马基雅维利主义 + 精神病态`：你既会看筹码，也更敢下手。和单纯强硬不同，你更像会先观察、再在关键时刻果断出牌的人。
- `自恋 + 精神病态`：你既不喜欢被轻视，也不太愿意在压力里退让。被冒犯、被忽略、被压住时，你的反应往往更直接、更硬。
- `马基雅维利主义高 + 其余两维中段`：你的核心不是表现欲或冲动，而是很会权衡关系、信息与时机。
- `自恋高 + 其余两维中段`：你的核心不是算计或攻击，而是明显需要被认可、被看见、被认真对待。
- `精神病态高 + 其余两维中段`：你的核心不是想控场或博关注，而是在冲突和压力里更不容易退、也更敢顶。

### 3.4 长图模块稿（适合分享与导出）

#### 3.4.1 长图目标

- 让用户截图或保存后，单看长图也能明白自己测出了什么
- 让朋友在社交平台看到时，第一眼能被 `标题 + 图谱 + 金句` 吸引
- 让长图既能承载内容厚度，又不至于像论文截图

#### 3.4.2 长图推荐结构

建议控制为 `7 ~ 9 个模块`，按“先结论、再拆解、后升华”的节奏排布：

1. `封面结论`
2. `三维图谱`
3. `主导维度解析`
4. `另外两维补充`
5. `人际模式`
6. `竞争模式`
7. `代价与提醒`
8. `分享页尾`

#### 3.4.3 长图模块详细稿

##### 模块 1｜封面结论

- 模块标题：`你的暗黑三角结果`
- 必备内容：`主导维度标题`、`一句话摘要`、`三枚维度标签`
- 推荐展示：大标题 + 副标题 + 主标签徽章
- 文案重点：先给出最能让用户记住的结论，不要一上来就讲定义

可直接使用的内容结构：

- 主标题：`你的结果更偏向「{主导维度副标题}」`
- 副标题：`{一句话摘要}`
- 标签行：`{标签1} / {标签2} / {标签3}`

##### 模块 2｜三维图谱

- 模块标题：`你的三维结构，不只是一句标签`
- 必备内容：`雷达图或三柱图`、`三维分数`、`高低排序`
- 推荐展示：图表在上，分数表在下
- 文案重点：让用户看懂自己不是“单一坏人设”，而是一种结构组合

可直接使用的辅助文案：

- `你的最高维是 {A}，第二高维是 {B}，这意味着你的表现并不是单线条的。`
- `如果两维分差很小，更适合把你理解为混合倾向，而不是单一标签。`

##### 模块 3｜主导维度解析

- 模块标题：`你最突出的那一面`
- 必备内容：`结果摘要`、`优势面`、`代价面`、`风险提醒`
- 推荐展示：大卡片，视觉等级最高
- 文案重点：既要让用户有被说中的感觉，也要避免把高分浪漫化

推荐卡片结构：

- `一句判断`
- `你通常会怎么处理局面`
- `为什么这会形成你的优势`
- `为什么这也会带来代价`

##### 模块 4｜另外两维补充

- 模块标题：`除了主导维度，你还有这两面`
- 必备内容：`次高维 + 最低维` 的分档解释
- 推荐展示：双卡并排或上下堆叠
- 文案重点：补足用户对“自己为什么不是纯一种风格”的理解

推荐写法：

- `第二高维决定你在关键时刻会不会“再多做一步”。`
- `最低维决定你更少用哪种方式处理关系和冲突。`

##### 模块 5｜人际模式

- 模块标题：`你在人际关系中的处理方式`
- 必备内容：`信任方式`、`边界方式`、`受伤后的反应`
- 推荐展示：三条短句 + 一个总结段
- 文案重点：把抽象特质翻成真实互动感

建议拆成三行：

- `你如何建立信任`
- `你如何保护自己`
- `你被冒犯时最可能的反应`

##### 模块 6｜竞争模式

- 模块标题：`你在利益与竞争场景里的风格`
- 必备内容：`拿资源的方式`、`是否爱出头`、`是否敢冒险`
- 推荐展示：关键词标签 + 2 段解释
- 文案重点：这是用户最容易觉得“准”的模块之一，建议写得更具体

建议覆盖的问题：

- `你更擅长控局、出头，还是硬顶？`
- `你会不会为了赢而放大手段？`
- `你更怕输面子，还是更怕输局面？`

##### 模块 7｜代价与提醒

- 模块标题：`这份倾向最容易带来的代价`
- 必备内容：`关系代价`、`情绪代价`、`现实后果`
- 推荐展示：警示卡风格，但不要做成恐吓式文案
- 文案重点：让用户觉得这份报告不是在夸他“暗黑很酷”，而是真的在做拆解

推荐结构：

- `如果走高了，最容易失去什么`
- `如果压久了，最容易在哪爆出来`
- `你最该警惕的不是别人，而是自己的哪种惯性`

##### 模块 8｜分享页尾

- 模块标题：`一句带走你的结果`
- 必备内容：`分享短句`、`边界说明`、`品牌或题集名`
- 推荐展示：适合单独截屏的收尾卡
- 文案重点：要利于分享，但不要像朋友圈鸡汤

页尾固定补一句：

- `这是一份人格倾向报告，不是临床诊断，也不等于道德评判。`

#### 3.4.4 长图内容取舍建议

如果长图需要做轻量版，优先保留：

- `封面结论`
- `三维图谱`
- `主导维度解析`
- `人际模式`
- `代价与提醒`
- `分享页尾`

如果做完整版，再补：

- `另外两维补充`
- `竞争模式`

#### 3.4.5 长图文案节奏建议

- 每屏只讲 `1 个核心点`
- 每屏标题尽量短，最好 `8~14 字`
- 每屏正文尽量控制在 `50~110 字`
- 每屏至少要有 `1 个可截图记忆点`，例如标签、短句、排序、图谱

### 3.6 分享页短句素材

- 马基雅维利主义偏高：`我不是城府深，我只是习惯先看局势。`
- 马基雅维利主义 + 自恋双高：`我既要赢局面，也不想输掉位置感。`
- 马基雅维利主义 + 精神病态双高：`我不是在试探边界，我只是更习惯直接拿结果。`
- 自恋偏高：`我不是非要站 C 位，只是不想被随便忽略。`
- 自恋 + 马基雅维利主义双高：`我既在意被看见，也知道怎么把局面拉向自己。`
- 自恋 + 精神病态双高：`我不只想赢注意力，也不怕把姿态放硬。`
- 精神病态偏高：`我不是没感觉，只是不喜欢在压力面前先退。`
- 精神病态 + 马基雅维利主义双高：`我会先看局势，也敢在关键时刻把牌打出去。`
- 精神病态 + 自恋双高：`我既不爱退场，也不接受自己被轻慢。`
- 三维较均衡：`我不是某一种极端，我只是更复杂一点。`
