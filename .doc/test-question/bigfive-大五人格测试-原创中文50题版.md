# Big Five 大五人格测试｜题目、评分规则、结果内容文档

> 设计定位：基于公开可参考的 IPIP Big-Five 50 题结构，整理为适合商业产品落地的原创中文 50 题版。
>
> 本文档只保留后续页面设计与测试引擎真正需要的三部分：题目、评分规则、结果内容。研究来源放在末尾附录。

## 1. 题目

### 1.1 作答形式

- 适合形态：`5 点量表单选`
- 推荐题干语气：第一人称自述句，用户按“像不像我”作答
- 推荐选项文案：
  - `非常不像我`
  - `不太像我`
  - `一般`
  - `比较像我`
  - `非常像我`
- 分值映射：`1 / 2 / 3 / 4 / 5`
- 交互建议：
  - 单题大卡片 + 固定底部选项条
  - 不展示实时结果
  - 显示进度，但不显示维度名称
  - 全程混排五维题目，避免用户看出明显套路

## 2. 评分规则

### 2.1 五维结构

- `E`：外向性（Extraversion）
- `A`：宜人性（Agreeableness）
- `C`：尽责性（Conscientiousness）
- `N`：神经质 / 情绪敏感度（Neuroticism）
- `O`：开放性（Openness to Experience）

### 2.2 题目归类与反向计分

#### 外向性 E

- 正向题：`1 / 11 / 21 / 31 / 41`
- 反向题：`6 / 16 / 26 / 36 / 46`

#### 宜人性 A

- 正向题：`2 / 12 / 22 / 32 / 42 / 47`
- 反向题：`7 / 17 / 27 / 37`

#### 尽责性 C

- 正向题：`3 / 13 / 23 / 33 / 43 / 48`
- 反向题：`8 / 18 / 28 / 38`

#### 神经质 N

- 正向题：`4 / 14 / 24 / 34 / 39 / 44 / 49`
- 反向题：`9 / 19 / 29`

#### 开放性 O

- 正向题：`5 / 15 / 25 / 35 / 45 / 50`
- 反向题：`10 / 20 / 30 / 40`

### 2.3 计分方式

- 每题分值：`1 - 5`
- 反向题换算：`反向得分 = 6 - 原始得分`
- 每个维度总分：该维度所有题目的换算后得分之和
- 每个维度平均分：`维度总分 / 10`

### 2.4 前端展示分数建议

> 因为没有真实常模时，不建议展示“超过全国 xx% 用户”之类的百分位文案。

- 页面内部可将维度平均分转成视觉指数：
  - `displayScore = ((avg - 1) / 4) * 100`
  - 输出范围：`0 - 100`
- 该指数只用于可视化，不等同于常模百分位

### 2.5 结果分档建议

> 仅用于产品结果文案分层，不宣称为医学或临床 cut-off。

- `低区`：`1.0 - 2.4`
- `中区`：`2.5 - 3.5`
- `高区`：`3.6 - 5.0`

### 2.6 结果展示规则

- 大五结果应展示为`五个独立维度`，不强行收束成单一人格类型
- 首页只突出：
  - `最高维度`
  - `第二显著维度`
  - `最低维度`
- 不建议生成“四字母”“动物型”“人格代号”之类二次包装结果
- 如果后期要做更深的“子维度 / facet”报告，应另行上更长题量版本，不建议从 50 题里硬拆 30 个 facet

### 2.7 题量建议

> 这是产品落地时最关键的决策之一。

- `10 题（TIPI）`：过短，只适合问卷研究补充或极强时间限制场景，不适合做付费核心商品
- `20 题（Mini-IPIP）`：可做免费引流版或前测版，但维度信息密度偏低，结果容易显得单薄
- `50 题`：最适合当前项目的主版本，兼顾完成率、结果稳定性、可读性与商业可售性
- `100 - 120 题`：适合后期高客单“长报告 / facet 深读版”，但不适合当前 MVP 首发

## 3. 结果内容

### 3.1 结果页信息架构建议

- `模块 1｜首屏结论`
  - 测试名
  - 用户昵称 / 日期
  - 一句话轮廓总结
  - 最高维度 / 第二显著维度 / 最低维度
- `模块 2｜五维雷达图 / 柱状图`
  - 五维可视化
  - 每维一句简述
- `模块 3｜五维长报告`
  - 每个维度单独解释
  - 分高 / 中 / 低三档文案
- `模块 4｜关系与协作提示`
  - 人际风格
  - 工作 / 学习风格
  - 压力下的表现
- `模块 5｜成长建议`
  - 根据最高维度和最低维度各给一条建议
- `模块 6｜长图导出内容`
  - 适合分享的封面句
  - 五维图
  - 重点维度卡片

### 3.2 首屏总结生成规则

- 只描述`维度组合`，不制造新的“人格类型”
- 推荐模板：
  - `你的轮廓更偏向「高{primaryName} + 高{secondaryName}」，同时在「{lowestName}」维度相对更克制。`
  - `你在与人相处、做事推进和感受世界的方式上，呈现出比较鲜明的五维偏好。`
- 如果最高维度与第二维度分差小于 `0.2`，可并列展示

### 3.3 五维长报告版文案

#### 3.3.1 外向性 E

**高区**

- 你通常更愿意与外部世界发生连接。
- 你在社交场合比较容易进入状态，也更愿意表达、带动和推进互动。
- 在团队或关系中，你往往是那个让气氛流动起来的人。
- 你可能会从交流、行动和外部反馈中获得能量。
- 提醒：高外向不等于一定擅长深关系经营，热闹不代表真正被理解。

**中区**

- 你在独处和社交之间通常能找到平衡。
- 需要表达时你能站出来，不需要时也能安静退后。
- 你不会为了热闹而热闹，也不会因为人多就完全退场。
- 这种状态往往让你在不同场景中都比较好适应。

**低区**

- 你更习惯把注意力放回自己内部，而不是持续向外释放。
- 你可能更偏好少量、稳定、有边界感的连接方式。
- 在新的群体环境中，你通常先观察，再决定是否投入。
- 这并不等于你不擅长关系，而是你更重视交流质量而不是数量。
- 提醒：当需求和想法长期不表达时，别人可能误以为你不在意。

#### 3.3.2 宜人性 A

**高区**

- 你通常会优先考虑关系感受，比较容易共情、体谅和照顾他人。
- 你在互动中倾向于缓和冲突，而不是扩大对立。
- 在亲密关系和合作关系里，你更容易被认为“好相处”。
- 你会在意别人是否舒服、是否被看见、是否被尊重。
- 提醒：过度迁就可能让你忽略自己的边界与真实需要。

**中区**

- 你既能理解别人，也不会完全失去自己的立场。
- 面对分歧时，你更可能先沟通再判断，而不是一味妥协。
- 这种平衡让你在人际中既不显得过硬，也不容易被裹挟。

**低区**

- 你更看重事实、原则、效率或个人判断，而不是先照顾所有人的感受。
- 你在冲突中可能更直接，也更愿意指出问题所在。
- 在某些需要决断或谈判的场景里，这会成为你的优势。
- 但在人际关系中，过于锋利的表达可能会被理解为冷淡、挑剔或难接近。
- 提醒：表达观点很重要，表达方式同样重要。

#### 3.3.3 尽责性 C

**高区**

- 你通常更有计划感、完成感和责任感。
- 面对任务时，你倾向于提前安排、按步骤推进，并在细节上保持稳定。
- 别人更容易把“靠谱”“可交付”“能落地”这些评价放在你身上。
- 你通常不喜欢失控、混乱或长期悬而未决的状态。
- 提醒：高尽责的人容易对自己要求过高，也容易因为偏差而自我施压。

**中区**

- 你会在秩序与弹性之间做调节。
- 该认真时能进入状态，遇到变化时也不至于完全僵住。
- 你未必是最严格执行计划的人，但通常具备基本的自我管理能力。

**低区**

- 你可能更偏向随势而动，而不是事事提前规划。
- 这会让你在灵活应变、临场反应和开放尝试上更轻盈。
- 但在长期任务、细节管理和稳定交付上，可能更容易出现拖延、遗漏或节奏失控。
- 提醒：当你不是没有能力，而是缺少结构时，结果很容易看起来“不够认真”。

#### 3.3.4 神经质 N

**高区**

- 你的情绪系统对压力、风险和不确定性更敏感。
- 你可能更容易担心、紧张、波动，也更容易被细微变化触发感受。
- 这种敏感并不全是缺点，它也意味着你更容易察觉问题和潜在风险。
- 但如果缺乏调节机制，你可能会更容易陷入反复内耗。
- 提醒：高神经质并不等于脆弱，而是说明你的感受阈值更低、反应更快。

**中区**

- 你的情绪反应通常处在可调节范围内。
- 你会受影响，但不至于轻易失控；会担心，但通常还能拉回理性。
- 这种状态有利于兼顾敏感度与稳定性。

**低区**

- 你整体更不容易被焦虑、低落或短期波动持续拖住。
- 面对变化时，你通常更能保持镇定和恢复力。
- 别人可能会觉得你情绪稳定、抗压、没那么容易被带着走。
- 提醒：稳定是一种优势，但过度压低感受也可能让你忽略自己真实的疲惫或需求。

#### 3.3.5 开放性 O

**高区**

- 你通常对新鲜经验、抽象问题、不同观点和想象空间更有兴趣。
- 你更愿意探索“还可以怎样”，而不只是接受既有答案。
- 你可能在内容审美、概念理解、创意连接或自我反思上更有能量。
- 在学习、创作或世界观形成上，你往往更需要精神层面的刺激。
- 提醒：高开放带来丰富视角，也可能带来犹豫、分散或难以落地。

**中区**

- 你既能接受新东西，也不会过度排斥稳定路径。
- 面对未知时，你会结合兴趣与现实判断是否投入。
- 这使你在创新与执行之间有一定平衡。

**低区**

- 你更偏好清晰、熟悉、可验证的方式，而不是无限发散。
- 你通常更看重实用性、确定性和现实可操作性。
- 在需要长期执行、流程稳定或标准明确的事情上，这会很有优势。
- 提醒：当环境变化很快时，过度依赖既有经验可能让你错过新的可能性。

### 3.4 页面可复用文案模块

#### 模块 1｜一句话轮廓

- `你整体呈现出较高的 {primary} 与 {secondary}，在 {lowest} 维度相对更克制。`
- `你的结果不是某一种“固定人格”，而是一张五维偏好地图。`

#### 模块 2｜五维标签短句

- 外向性：`你更倾向从外部互动中获得能量。`
- 宜人性：`你在关系中更重视理解、体谅与合作。`
- 尽责性：`你对秩序、责任与完成度比较敏感。`
- 神经质：`你对压力与情绪波动的感受阈值更高 / 更低。`
- 开放性：`你对新观点、新体验与复杂问题更有兴趣。`

#### 模块 3｜成长建议生成逻辑

- 若 `primary = C`：建议加入“留出弹性窗口，避免对偏差过度苛责自己”
- 若 `primary = A`：建议加入“照顾关系的同时，保留边界表达”
- 若 `primary = O`：建议加入“把想法拆成可执行步骤，减少只停留在灵感层”
- 若 `primary = E`：建议加入“在表达很多时，也给自己留出真正消化感受的空间”
- 若 `primary = N`：建议加入“先区分真实风险与想象风险，再决定要不要投入情绪”

- 若 `lowest = C`：补充“建立外部结构比单靠意志力更有效”
- 若 `lowest = A`：补充“练习在表达事实时照顾关系感受”
- 若 `lowest = O`：补充“在稳定路径之外，给自己一点试错额度”
- 若 `lowest = E`：补充“在重要关系里适度表达，比沉默更容易被理解”
- 若 `lowest = N`：补充“稳定不是压抑，记得识别并回应自己的情绪信号”

### 3.5 长图模块稿（适合导出分享）

#### 模块 1｜封面

- 标题：`你的大五人格轮廓`
- 副标题：`Big Five Personality Profile`
- 结论短句：`高 {primary} · 高 {secondary} · 相对低 {lowest}`

#### 模块 2｜五维图谱

- 雷达图或横向柱状图
- 每个维度标注：`低 / 中 / 高`

#### 模块 3｜显著维度解读

- 放最高维度的 3 条核心描述
- 放第二显著维度的 2 条补充描述

#### 模块 4｜低位维度提醒

- 说明该维度较低时的日常表现
- 给出 1 条非说教式建议

#### 模块 5｜关系 / 工作 / 压力

- 关系：你在人际中更容易呈现什么样的互动方式
- 工作：你在任务推进中更自然的节奏
- 压力：你在高压时更容易出现什么状态

#### 模块 6｜页尾分享语

- `这不是给你贴标签，而是帮你看见自己的稳定偏好。`

### 3.6 分享短句素材

### 3.7 结果页模块详细稿（增强版）

#### 模块 1｜首屏 Hero

- 主标题：`你的大五人格画像`
- 副标题：`不是把你塞进某一型，而是还原你在五个维度上的稳定偏好。`
- 核心信息区建议包含：
  - `最高维度`
  - `第二显著维度`
  - `最低维度`
  - `一句话轮廓`
- 一句话轮廓示例：
  - `你整体呈现出高宜人性、高开放性，在神经质维度相对更低。你通常更容易理解他人，也更愿意探索复杂和新鲜的事物。`
  - `你的结果显示出较高尽责性与较低神经质，这意味着你在任务推进和压力恢复上通常更稳定。`

#### 模块 2｜五维总览区

- 推荐组件：`五维雷达图 + 五张维度摘要卡`
- 每张卡片固定包含：
  - 维度名
  - 高 / 中 / 低标签
  - 1 句总述
  - 1 个优势词
  - 1 个提醒词
- 维度摘要卡短文案示例：
  - 外向性：`你更偏向主动连接，还是更习惯保留内在空间。`
  - 宜人性：`你在关系里更看重体谅合作，还是更强调判断与立场。`
  - 尽责性：`你对秩序、节奏和完成度的要求有多高。`
  - 神经质：`你对压力、风险与情绪波动的敏感度如何。`
  - 开放性：`你面对新观点、新体验和复杂问题时的开放程度。`

#### 模块 3｜关系画像区

- 目标：回答用户最关心的现实问题——`我在人际和亲密关系里会是什么样子？`
- 建议展示 3 个子模块：
  - `你给人的第一印象`
  - `你在关系中的舒适区`
  - `你容易被误解的地方`
- 示例生成逻辑：
  - `高 E + 高 A`：更容易给人热情、亲近、可互动的第一印象
  - `低 E + 高 O`：更容易被误解为疏离，但实际上更重视精神层面的匹配
  - `低 A + 高 C`：容易被视为严格、理性、标准高，但并不等于没有善意
  - `高 N`：更容易在关系里表现出敏感、确认需求强、对细节更在意

#### 模块 4｜工作 / 学习画像区

- 目标：把人格结果转成更有“购买价值”的现实解释
- 建议展示 4 个标签：
  - `任务启动`
  - `执行节奏`
  - `协作偏好`
  - `容易卡住的点`
- 示例：
  - `高 C`：启动后更容易稳定推进，但也可能过度追求完成质量
  - `高 O`：创意、理解、联想能力更强，但有时会停留在构想阶段
  - `高 E`：更适合在互动反馈中推进任务
  - `低 E`：更适合在安静、不被频繁打断的环境里深度处理信息

#### 模块 5｜压力状态区

- 目标：告诉用户“状态差时我通常会怎么表现”
- 建议分 3 层：
  - `压力来临时`
  - `你最容易掉进的循环`
  - `更适合你的恢复方式`
- 示例生成逻辑：
  - `高 N`：更容易提前担心、反复想、情绪卷入更快
  - `低 N`：更能稳住表面状态，但也可能延迟察觉真正疲惫
  - `高 C`：压力下容易加码控制
  - `高 E`：恢复更依赖与人连接和外部能量
  - `低 E`：恢复更依赖独处、降噪和内部整理

#### 模块 6｜优势与盲区区

- 每份报告都建议固定输出：
  - `你的天然优势`
  - `你可能忽略的盲区`
  - `一句成长提醒`
- 示例模板：
  - `你的高开放让你比很多人更容易看见复杂性，但也可能因此更难快速收束判断。`
  - `你的高宜人性让你很会照顾关系，但也容易把体谅别人变成压缩自己。`
  - `你的低神经质是稳定优势，但别因为“看起来没事”就忽略自己的真实消耗。`

#### 模块 7｜页尾结语区

- 适合放 2 类文案：
  - `反标签化提醒`：`这不是对你下定义，而是帮你识别自己的稳定偏好。`
  - `行动建议`：`理解最高维度很重要，看见最低维度同样重要。它往往决定了你在哪些场景更容易卡住。`

### 3.8 五维组合摘要模板库

#### 高 C + 低 N

- `你通常给人稳定、靠谱、能托付事情的感觉。`
- `你不太容易在波动中失控，也更愿意用结构和计划稳住局面。`
- `你的优势是抗压与执行，但要留意别把自己活成只会解决问题的人。`

#### 高 O + 低 E

- `你未必热闹，但通常很有自己的精神世界。`
- `你更容易被复杂内容、独特观点和深度连接吸引，而不是被表面的活跃氛围吸引。`
- `你的问题通常不是没内容，而是内容不一定第一时间被别人看见。`

#### 高 A + 高 E

- `你通常既有亲和力，也有互动感。`
- `你更容易在群体里建立连接，也更容易让别人感到被接纳。`
- `要留意的一点是：你习惯照顾气氛时，可能会忽略自己真正想表达什么。`

#### 高 O + 高 C

- `你既有想法，也有把想法落地的潜力。`
- `相比只会发散或只会执行的人，你更可能同时具备理解复杂问题与持续推进的能力。`
- `你的挑战在于：标准太高时，容易在“要不要更好一点”里拖慢节奏。`

#### 高 N + 高 A

- `你对关系和感受都很敏锐。`
- `你容易在意别人，也容易被互动细节触动。`
- `这种细腻会让你更会照顾关系，但也更容易因为别人的反应而反复消耗自己。`

#### 低 A + 高 C

- `你通常更讲原则、效率和标准，而不是只顾及场面上的舒服。`
- `在需要判断、推进和定规则的场景里，这是一种优势。`
- `但在关系里，你的正确很可能需要一个更柔和的表达出口。`

### 3.10 长图模块扩展建议（50 题版）

- `第 1 屏`：封面结论 + 五维标签
- `第 2 屏`：五维雷达图 + 一句话解释
- `第 3 屏`：最高维度详解
- `第 4 屏`：第二显著维度详解
- `第 5 屏`：最低维度提醒
- `第 6 屏`：关系画像
- `第 7 屏`：工作 / 学习画像
- `第 8 屏`：压力状态 + 恢复建议
- `第 9 屏`：成长提醒 + 分享页尾

### 3.11 长图文案密度建议

- 50 题版结果页要避免“像免费小测试一样只有几句空话”
- 每个主要维度卡建议至少包含：
  - `1 条总结`
  - `2 条具体表现`
  - `1 条提醒`
- 整份长图建议控制在：
  - `8 - 10 屏`
  - `1200 - 2200 字`
  - 兼顾“读起来值回票价”和“适合手机截图分享”
