## 🧪 评分机制设计

### 维度模型

采用 **单维度累加模型 + 五维成分拆解**：

- **主维度：班味浓度值** — 从「几乎无味」到「浓度爆表」，越高代表被工作腌得越入味
- 每道题 4 个选项，分值为 1-4 分（1 = 班味最淡，4 = 班味最浓）

### 五维成分拆解

15 道题分为 5 组（每组 3 题），分别对应班味的 5 种「化学成分」，用于生成成分分析表：

| 成分名称 | 对应题号 | 检测内容 |
| --- | --- | --- |
| ☕ 咖啡因依赖度 | Q1, Q6, Q11 | 身体疲劳 & 精神状态 |
| 📊 KPI 渗透率 | Q2, Q7, Q12 | 工作思维入侵日常 |
| 🫠 精神内耗浓度 | Q3, Q8, Q13 | 心理状态 & 情绪损耗 |
| 📱 工位依赖症 | Q4, Q9, Q14 | 行为模式 & 习惯异化 |
| 🌙 生活褪色指数 | Q5, Q10, Q15 | 个人生活被工作侵蚀程度 |

每个成分的得分范围为 3-12 分，换算为百分比后展示在「成分分析表」中。

### 计分方式

15 道题做完后累加总分：

- 总分范围：**15 - 60 分**
- 换算为浓度百分比：浓度 = (总分 - 15) / 45 × 100%

### 浓度等级映射

| 等级 | 总分区间 | 浓度范围 | 等级名称 | 一句话描述 |
| --- | --- | --- | --- | --- |
| Lv.1 | 15 - 22 分 | 0% - 15% | 🫧 班味微量 | 恭喜，你还是个人 |
| Lv.2 | 23 - 30 分 | 16% - 33% | 🧊 班味初现 | 你开始有点味道了 |
| Lv.3 | 31 - 38 分 | 34% - 51% | 🧂 班味适中 | 标准打工人配方 |
| Lv.4 | 39 - 46 分 | 52% - 69% | 🫗 班味浓郁 | 你已经被腌入味了 |
| Lv.5 | 47 - 54 分 | 70% - 87% | 🔥 班味爆表 | 你不是在上班，你是班本班 |
| Lv.6 | 55 - 60 分 | 88% - 100% | ☢️ 班味成精 | 工作已经成为你的DNA |

<aside>
💡

**设计逻辑**：6 级划分让结果有层次感。低分段（Lv.1-2）是「轻松调侃」，中间段（Lv.3-4）覆盖大部分用户，高分段（Lv.5-6）制造「震撼感」和「炫耀式自嘲」。五维成分表增加报告的专业感和可读性，也让用户更愿意截图分享。

</aside>

<aside>
🎯

**结果页设计原则**

- 采用 **实验室检测报告** 视觉风格：试管刻度、成分分析表、医嘱单
- 每个等级都是 **搞笑 + 共鸣** 的，没有真正让人不舒服的结果
- 内容模块：检测编号 + 浓度等级试管 + 成分分析表 + 检测概述 + 症状描述 + 临床表现 + 并发症 + 医嘱建议 + 同类标本 + 解药配方 + 专属打工语录 + 最佳共鸣搭子
- 低浓度 = 调侃「你怎么这么清醒」，高浓度 = 「打工人 DNA 共鸣到飞起」

### 🫧 Lv.1 班味微量（15-22 分 / 浓度 0%-15%）

**检测编号**：BW-2026-LV1-PURE

**「恭喜您，检测结果显示：您几乎不含班味成分。请问……您是怎么做到的？」**

#### 🧪 检测概述

您的样本在实验室标准检测流程中几乎未检出班味成分。仪器多次确认后，实验员开始怀疑是不是搞错了样本。您的存在让整个实验室陷入了沉思——在这个人均班味浓郁的时代，您是怎么保持如此纯净的？

#### 🔬 症状描述

- 周一和周五的精神状态没有明显差异
- 能准时下班且毫无心理负担
- 工作群消息只在工作时间查看，下班后完全无视
- 拥有持续运转的爱好、社交和个人生活
- 对"内卷"一词感到困惑：不是所有人都到点下班吗？

**关键标签**：`#人间清醒` `#职场边界大师` `#准时下班冠军` `#生活优先` `#班味绝缘体`

#### 📋 临床表现

- **上班时**：准时到岗，高效完成工作，到点走人，绝不磨洋工也绝不加班
- **下班后**：手机调静音，进入"与工作无关的美好人生"模式
- **周末时**：什么工作？什么工位？我只知道今天要去哪里玩
- **假期时**：从不带电脑出门，也从不在度假时"看一眼消息"

#### ⚠️ 并发症

- **同事不理解综合征**：别人加班时你准点走，收获异样目光若干
- **莫名的罪恶感**：偶尔怀疑自己是不是太佛系了，但转念一想——"管他呢"
- **精神状态过好引发嫉妒症**：朋友和同事看你的眼神逐渐变成"你到底在哪上班？能把我也弄进去吗"

#### 💊 医嘱建议

请继续保持。您是职场中的稀有物种，请保护好自己的精神生态。您可能不需要这份检测报告的任何建议——事实上，您应该反过来给我们提建议。

**唯一注意事项**：偶尔关心一下同事的情绪，因为您的从容可能正在无形中对他们造成精神伤害。

#### 🧫 同类标本

- 欧洲打工人（准时下班是写进法律的那种）
- 国企养老岗位传说中的存在
- 财务自由后还在上班纯为了交社保的人

#### 💉 解药配方

您不需要解药。事实上，您本身就是解药——请把您的 work-life balance 秘方分享给身边浓度超标的朋友。

#### 🗣️ 专属打工语录

> "工作是为了更好地生活，不是反过来。"

### 🧊 Lv.2 班味初现（23-30 分 / 浓度 16%-33%）

**检测编号**：BW-2026-LV2-MILD

**「检测到微量班味成分。别紧张，你还有救。」**

#### 🧪 检测概述

您的样本中检出了微量班味成分，目前处于「入味初期」。就像泡菜刚放进坛子的第一天——表面上看起来还是新鲜蔬菜的样子，但如果你仔细闻，已经能闻到一丝丝发酵的气息了。好消息是：现在抢救还来得及。

#### 🔬 症状描述

- 周一有点不想起床，但周五精神百倍
- 下班后偶尔会刷一下工作消息，但能控制住不回
- 开始积累了一些职场口头禅，但还分得清什么时候该说人话
- 偶尔在周日晚上感到一丝莫名的焦虑
- 工作和生活还有明确的边界，只是这条线开始微微变模糊了

**关键标签**：`#班味入门` `#清醒中带点迷糊` `#周一恐惧轻症` `#还能抢救` `#泡菜初期`

#### 📋 临床表现

- **上班时**：能正常工作，偶尔摸鱼无罪恶感，整体节奏还不错
- **下班后**：大部分时间属于自己，但偶尔会心虚地瞄一眼工作群
- **周末时**：大多数时候能开心玩耍，但周日傍晚那个"哎"的叹气越来越响了
- **假期时**：前两天完全释放，第三天开始偶尔想起工作

#### ⚠️ 并发症

- **"我是不是太卷了"自我怀疑症**：偶尔加了一次班就开始反思人生
- **职场语言初期感染**：不小心对朋友说了句"我们对齐一下"，然后两个人一起沉默了三秒
- **周五狂欢综合征**：一到周五下午就开始精神抖擞，被同事评价为"判若两人"

#### 💊 医嘱建议

您目前处于班味可控阶段，建议加强个人边界建设。具体处方：

- 下班后手动关闭工作 App 通知（物理隔离法）
- 每周至少安排 1 次"纯粹因为快乐而做的事"
- 如果周日开始焦虑，请立刻做一件让自己开心的事打断它

**预后评估**：如不加干预，6 个月后预计升级至 Lv.3。请珍惜您当前的清醒时刻。

#### 🧫 同类标本

- 入职 1-2 年的年轻打工人
- 刚从"工作好有趣"过渡到"工作还行吧"的人

  caffeine: 0,    // 咖啡因依赖度（Q1, Q6, Q11）

};

  0: 'caffeine', 5: 'caffeine', 10: 'caffeine',
  1: 'kpiInvasion', 6: 'kpiInvasion', 11: 'kpiInvasion',
  2: 'mentalDrain', 7: 'mentalDrain', 12: 'mentalDrain',
  3: 'deskBond', 8: 'deskBond', 13: 'deskBond',
  4: 'lifeFade', 9: 'lifeFade', 14: 'lifeFade',
};

// 每道题选择后累加分值
function onAnswer(questionIndex, score) {
  totalScore += score;
  const dim = questionDimensionMap[questionIndex];
  dimensions[dim] += score;
}

// 计算浓度百分比
function getConcentration(total) {
  return Math.round((total - 15) / 45 * 100);
}

// 计算单维度百分比（每维度 3 题，范围 3-12）
function getDimensionPercent(dimScore) {
  return Math.round((dimScore - 3) / 9 * 100);
}

// 判定浓度等级
function getLevel(total) {
  if (total <= 22) return 'lv1';
  if (total <= 30) return 'lv2';
  if (total <= 38) return 'lv3';
  if (total <= 46) return 'lv4';
  if (total <= 54) return 'lv5';
  return 'lv6';
}
