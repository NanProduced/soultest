## 1. 选题核心逻辑

<aside>
🎯

**爆款测试的三要素**

- **钩子力**：标题一眼抓住注意力，激发"我也想测"的冲动
- **情绪价值**：结果让用户感到被"看见"、被"理解"、被"夸赞"（巴纳姆效应）
- **传播因子**：结果页本身就是一张"值得发朋友圈"的精美图片
</aside>

### 1.1 小红书用户的六大情绪密码

1. **🔍 自我探索欲** — "原来我是这样的人！" → MBTI、九型人格、黑暗三角
2. **🏷️ 标签认同感** — "这个标签太准了！" → 城市匹配、动物人格、颜色性格
3. **💕 关系映射** — "难怪我们合不来/很合拍！" → 爱情语言、依恋类型、CP 匹配
4. **🌟 正向肯定** — "我原来这么优秀！" → 天赋测试、隐藏能力、前世今生
5. **😈 暗黑猎奇** — "我的暗黑面是什么？" → 黑暗人格、七宗罪、反社会指数
6. **🎭 娱乐社交** — "哈哈哈这也太搞了！" → 社死指数、恋爱脑指数、发疯文学

### 1.2 品类优先级判断标准

| 维度 | 权重 | 说明 |
| --- | --- | --- |
| **市场验证度** | ⭐⭐⭐⭐⭐ | 小红书/抖音上已有爆款先例，证明用户愿意为此付费 |
| **情绪价值** | ⭐⭐⭐⭐⭐ | 结果是否能让用户产生强烈的认同感、惊喜感或分享欲 |
| **分享传播性** | ⭐⭐⭐⭐ | 结果是否天然适合做成精美长图分享到社交平台 |

| **UI 差异化空间** | ⭐⭐⭐⭐ | 竞品体验是否差，我们是否能通过 UI/UX 拉开差距 |

## 2. 七大品类深度分析

### 2.1 🧠 性格人格类（核心品类，必做）

**市场表现**：小红书 MBTI 相关笔记 40 万+，话题浏览量 22.8 亿+。"言策的店"仅靠性格测试就售出 6.6 万件。

**爆点逻辑**：巴纳姆效应 + 标签化认同 + 社交货币（"你测了吗？你是什么型？"）

**经典题集**：

- **MBTI（16型人格）**：最经典，永不过时。93 题标准版 / 精简 28 题版 / 12 题极速版
- **九型人格（Enneagram）**：144 题完整版 / 45 题精简版，9 种人格分类
- **DISC 性格测试**：40 题，D 支配/I 影响/S 稳健/C 谨慎，职场向
- **大五人格（Big Five/OCEAN）**：开放性/尽责性/外向性/宜人性/神经质，学术最认可
- **霍兰德职业兴趣**：120 题完整版 / 60 题精简版，6 种职业类型

**我们的 UI 差异化**：

- 竞品全是纯文字单选 → 我们用**图片选择 + 情景模拟 + 滑块量表**混合题型
- 竞品结果页纯文字 → 我们用**雷达图 + 匹配度圆环 + 标签云 + 精美配色**
- 竞品无分享功能 → 我们**一键导出带店铺二维码的精美长图**

---

### 2.2 🏙️ 匹配类（高传播，必做）

**市场表现**："你的灵魂城市""你的 MBTI 动物"等匹配类测试是小红书销量最高的品类之一。

**爆点逻辑**：将抽象性格具象化为大家熟悉的事物（城市/动物/颜色/食物），天然具备"分享标签"属性。

**热门主题**：

- **灵魂城市匹配**："你的灵魂属于哪座城市？" — 巴黎/东京/纽约/成都/墨尔本…
- **MBTI 动物人格**：将 16 种 MBTI 对应 16 种动物，双重标签
- **灵魂颜色/色彩人格**："你的灵魂是什么颜色？" — 视觉冲击力最强
- **灵魂花朵/植物**：文艺清新风，特别适合小红书女性用户
- **灵魂季节**：春夏秋冬四季对应不同性格
- **专属 BGM/歌曲匹配**："你的人生 BGM 是什么？"

**UI 差异化**：匹配类最适合做**视觉炸裂**的结果页 — 城市实景背景 + 半透明卡片 + 匹配度动画

---

### 2.3 💕 情感关系类（复购率高）

**市场表现**："爱情语言""依恋类型"在小红书持续有热度，用户经常拉伴侣一起测。

**爆点逻辑**：关系是永恒话题，"原来我们不合是因为这个！"的恍然大悟感极强。

**经典题集**：

- **五种爱情语言**：Gary Chapman 的经典理论 — 肯定言辞/服务行为/接收礼物/优质时间/身体接触
- **依恋类型测试**：安全型/焦虑型/回避型/恐惧型，亲密关系核心理论
- **恋爱人格/动物恋爱型**：金毛型/猫咪型/狐狸型等，轻松有趣
- **伴侣匹配度测试**：两人各测一次，对比结果看匹配度
- **爱情观测试**：浪漫主义/现实主义/理想主义/实用主义

**UI 差异化**：双人模式 — 支持两人各自答题后合并出匹配报告，结果页用**对比雷达图 + 匹配度百分比**

---

### 2.4 😈 暗黑系（引流利器，高话题度）

**市场表现**：36kr 报道"0.99 元黑暗人格测试刷屏，有人靠它月入过万"。暗黑系天然具备猎奇吸引力。

**爆点逻辑**：人人都好奇自己的"阴暗面"，但结果不会太负面（巧妙包装为"独特个性"）。

**热门主题**：

- **黑暗三角人格**：自恋/马基雅维利主义/精神病态，经典心理学量表
- **七宗罪测试**：傲慢/贪婪/淫欲/嫉妒/暴食/愤怒/懒惰，7 个维度
- **反社会指数**："你有多反社会？" — 刺激但无害的猎奇感
- **黑暗超能力**："如果你是反派，你的超能力是？" — 趣味包装
- **暗黑 MBTI**：每种人格的阴暗面解读

**UI 差异化**：暗黑系适合**深色主题 + 霓虹发光效果 + 神秘感十足的动画**，与竞品的廉价 H5 感形成鲜明对比

---

### 2.5 🌟 正能量/天赋类（高情绪价值）

**爆点逻辑**：纯正向肯定，"原来我这么厉害"的惊喜感。用户分享欲最强，因为结果全是夸自己。

**热门主题**：

- **隐藏天赋测试**："你不知道的自己的 5 大天赋"
- **前世今生**："你的前世是什么身份？" — 公主/将军/诗人/探险家
- **灵魂年龄**："你的灵魂几岁？" — 老灵魂 vs 年轻灵魂
- **人生主题词**："定义你 2025 的三个关键词" — 适合年初/年末
- **超能力匹配**："如果你有超能力，你会拥有什么？"

---

### 2.6 🎭 趣味娱乐类（引流 + 社交传播）

**爆点逻辑**：轻松搞笑，无心理压力，适合做低价引流款或免费试玩款。

**热门主题**：

- **社死指数**："你的社死等级有多高？" — 场景题超有代入感
- **恋爱脑指数**："你有多恋爱脑？" — 扎心但好笑
- **发疯文学等级**："你离发疯还有几步？" — 当代年轻人共鸣
- **精神状态检测**："你现在的精神状态像哪种动物？" — 社交热梗
- **i 人/e 人极限测试**："你到底有多 i/e？" — MBTI 衍生轻量版
- **打工人疲惫指数**："你的班味有多重？" — 职场共鸣

---

### 2.7 🔮 玄学/命理类（高客单价潜力）

**爆点逻辑**：年轻人"不信但想看看"的心理，与星座文化高度重合。

**热门主题**：

- **塔罗牌性格**："你是哪张塔罗牌？" — 22 张大阿尔卡纳对应性格
- **星座深度解读**：超越基础星座，加入上升/月亮/金星等维度
- **生命数字/数字命理**：根据生日算出 1-9 生命数字
- **五行性格**：金木水火土五行对应性格特征

## 3. 首批上线建议（P0 优先级）

<aside>
🚀

**建议首批上线 5 套测试，覆盖不同品类**

1. **灵魂城市匹配** — 匹配类，视觉最炸裂，最适合展示 UI 差异化
2. **MBTI 精简版（28题）** — 性格类，搜索量最大，刚需
3. **黑暗三角人格** — 暗黑系，已验证的爆款，引流利器
4. **五种爱情语言** — 情感类，复购率高（拉伴侣/朋友一起测）
5. **社死指数 / i人e人极限测试** — 趣味类，低门槛引流款
</aside>
